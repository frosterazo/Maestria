Productos Notables: Atajos algebraicos clave

Nivel: 1° Año Medio
Eje: Álgebra
Duración: 90 min
Currículum: MINEDUC 2026
OA 3
Objetivo de Aprendizaje: Desarrollar los productos notables de manera concreta, pictórica y simbólica: transformar productos en sumas y viceversa, aplicarlos a situaciones concretas, completar el cuadrado del binomio, y utilizarlos en la reducción y desarrollo de expresiones algebraicas.
1
¿Qué son los productos notables?

Los productos notables son multiplicaciones algebraicas que aparecen tantas veces que vale la pena memorizar sus resultados. Son atajos que ahorran cálculo y son indispensables en factorización, ecuaciones cuadráticas y geometría.

Tres productos notables fundamentales
  1. Cuadrado de un binomio: (a + b)²  |  (a − b)²
  2. Suma por su diferencia: (a + b)(a − b)
  3. Producto de binomios con término común: (x + a)(x + b)
2
Cuadrado de un binomio
Cuadrado de la suma
(a + b)² = a² + 2ab + b²
Cuadrado de la diferencia
(a − b)² = a² − 2ab + b²

Regla mnemotécnica: "El cuadrado del primero, más o menos el doble producto del primero por el segundo, más el cuadrado del segundo".

Ejemplo — Desarrollar (x + 3)²
1
Cuadrado del primerox²
2
Doble producto2 · x · 3 = 6x
3
Cuadrado del segundo3² = 9
✓
Resultado(x + 3)² = x² + 6x + 9
⚠️Error común
(a + b)² ≠ a² + b²
Olvidar el doble producto es el error más común. Verifica con números: (2+3)² = 25, mientras que 2² + 3² = 13. ¡No es lo mismo!
3
Suma por su diferencia
Producto de suma por diferencia
(a + b)(a − b) = a² − b²

El resultado es siempre una diferencia de cuadrados. Los términos cruzados se cancelan.

Ejemplo — Desarrollar (x + 5)(x − 5)
1
Identificar a y ba = x, b = 5
2
Aplicar fórmula a² − b²x² − 5²
✓
Resultado(x + 5)(x − 5) = x² − 25
⚡Truco para cálculo mental
97 · 103 = (100 − 3)(100 + 3) = 100² − 3² = 10.000 − 9 = 9.991. ¡En segundos sin calculadora!
4
Binomios con término común
Producto con término común
(x + a)(x + b) = x² + (a + b)x + a·b

Regla: El cuadrado del término común, más la suma de los términos no comunes multiplicada por el término común, más el producto de los términos no comunes.

Ejemplo — (x + 2)(x + 5)
1
Cuadrado del común (x)x²
2
Suma de los no comunes por x(2 + 5)x = 7x
3
Producto de los no comunes2 · 5 = 10
✓
Resultado(x + 2)(x + 5) = x² + 7x + 10
5
Factorización: el camino inverso

Factorizar es el proceso inverso: pasar de una expresión desarrollada a un producto.

ReconocerFactorizar comoEjemplo
a² + 2ab + b²(a + b)²x² + 6x + 9 = (x + 3)²
a² − 2ab + b²(a − b)²x² − 10x + 25 = (x − 5)²
a² − b²(a + b)(a − b)x² − 49 = (x + 7)(x − 7)
x² + (a+b)x + ab(x + a)(x + b)x² + 5x + 6 = (x + 2)(x + 3)
6
Completación del cuadrado

Es transformar una expresión cuadrática del tipo x² + bx en un cuadrado de binomio, agregando un término que la complete.

Regla
Para completar x² + bx, suma (b/2)²
Ejemplo — Completar x² + 8x
1
Calcular (b/2)²(8/2)² = 4² = 16
2
Sumar al originalx² + 8x + 16
✓
Factorizarx² + 8x + 16 = (x + 4)²

Nivel Básico

Reconocimiento y aplicación directa

Ejercicio 1
Básico
¿Cuál es el desarrollo de (x + 4)²?
✓ Correcto. (x+4)² = x² + 2·(4)·x + 4² = x² + 8x + 16. El doble producto es 2·x·4 = 8x, no 4x.
✗ Incorrecto. (x+4)² = x² + 2·4·x + 4² = x² + 8x + 16 (opción B). El error más común es olvidar el doble producto o calcular solo la mitad de él.
Ejercicio 2
Básico
¿Cuál es el resultado de (x + 3)(x − 3)?
✓ Correcto. Suma por diferencia: (a+b)(a−b) = a² − b². Aquí a = x, b = 3 → x² − 9. Los términos cruzados se cancelan.
✗ Incorrecto. Esta es la fórmula suma por diferencia: (x+3)(x−3) = x² − 3² = x² − 9 (opción B). No hay términos cruzados porque +3x y −3x se cancelan.
Ejercicio 3
Básico
¿Cuál es el desarrollo de (x − 2)²?
✓ Correcto. (x−2)² = x² − 2·(2)·x + 2² = x² − 4x + 4. En el cuadrado de diferencia, el término medio es negativo, pero el último siempre es positivo (b² ≥ 0).
✗ Incorrecto. (x−2)² = x² − 2·2·x + 2² = x² − 4x + 4 (opción C). El último término siempre es positivo (2² = +4). El doble producto lleva el signo negativo del binomio.
Nivel Medio

Aplicación con expresiones complejas

Ejercicio 4
Medio
Desarrolla: (2x + 3)²
✓ Correcto. a = 2x, b = 3. (2x)² = 4x². 2·(2x)·3 = 12x. 3² = 9. Total: 4x² + 12x + 9.
✗ Incorrecto. (2x+3)²: a = 2x, b = 3. (2x)² = 4x² (no 2x²). Doble producto: 2·2x·3 = 12x. b² = 9. Resultado: 4x² + 12x + 9 (opción A).
Ejercicio 5
Medio
Factoriza: x² − 25
✓ Correcto. x² − 25 = x² − 5² es una diferencia de cuadrados → (x+5)(x−5). No es un cuadrado de binomio porque falta el término medio.
✗ Incorrecto. x² − 25 = x² − 5². Diferencia de cuadrados → (x+5)(x−5) (opción C). Si fuera (x−5)² = x² − 10x + 25, faltaría el término −10x.
Ejercicio 6
Medio
Desarrolla: (x + 4)(x + 6)
✓ Correcto. (x+a)(x+b) = x² + (a+b)x + ab. Aquí a=4, b=6: x² + (4+6)x + 4·6 = x² + 10x + 24.
✗ Incorrecto. (x+4)(x+6): el coeficiente medio es la suma (4+6=10) y el término independiente es el producto (4·6=24). Resultado: x² + 10x + 24 (opción B).
Nivel Avanzado

Factorización y aplicaciones

Ejercicio 7
Avanzado
Factoriza: x² + 10x + 25
✓ Correcto. a² + 2ab + b²: a = x, b = 5. Verificar: 2·5·x = 10x ✓ y 5² = 25 ✓. Resultado: (x+5)². Si el signo del medio fuera −10x, sería (x−5)².
✗ Incorrecto. x²+10x+25 = (x+5)² (opción A) porque 2·5·x=10x y 5²=25. No es diferencia de cuadrados (no es a²−b²). No es (x−5)² porque eso daría x²−10x+25.
Ejercicio 8
Avanzado
Factoriza: x² + 8x + 15
✓ Correcto. Buscar dos números que sumen 8 y multipliquen 15: 3 + 5 = 8 ✓ y 3 · 5 = 15 ✓. Por tanto: (x+3)(x+5).
✗ Incorrecto. Busca dos números que sumen 8 y cuyo producto sea 15: 3 y 5 cumplen (3+5=8 y 3·5=15). Resultado: (x+3)(x+5) (opción A). No son negativos porque el término +15 es positivo.
Ejercicio 9
Avanzado
Calcula mentalmente: 98 · 102
✓ Correcto. 98·102 = (100−2)(100+2) = 100² − 2² = 10.000 − 4 = 9.996. La fórmula suma por diferencia permite cálculo mental inmediato.
✗ Incorrecto. Aplica suma por diferencia: 98·102 = (100−2)(100+2) = 100²−2² = 10.000−4 = 9.996 (opción A). 10.000 sería 100² sin restar el 2².
Ejercicio 10
Avanzado
¿Qué término hay que sumar a x² + 12x para completar el cuadrado?
✓ Correcto. Regla: (b/2)². Aquí b=12 → (12/2)² = 6² = 36. Resultado: x²+12x+36 = (x+6)².
✗ Incorrecto. Para completar el cuadrado de x²+bx, se suma (b/2)². Con b=12: (12/2)² = 6² = 36 (opción C). No es 6 (que sería solo b/2) ni 12 ni 24.

Pregunta 1
Desarrolla: (x + 7)²
✓ Correcto. (x+7)² = x² + 2·7·x + 7² = x² + 14x + 49. El doble producto es 2·7 = 14, no 7.
✗ Incorrecto. (x+7)² = x² + 2·7·x + 49 = x² + 14x + 49 (opción B). La opción A olvida el doble producto; C toma solo la mitad.
Pregunta 2
Desarrolla: (3x − 2)²
✓ Correcto. (3x)² = 9x². Doble producto: −2·(3x)·2 = −12x. 2² = 4. Total: 9x² − 12x + 4.
✗ Incorrecto. (3x−2)²: (3x)²=9x², −2·3x·2=−12x (negativo), 2²=4. Total: 9x²−12x+4 (opción A). El término medio es negativo porque la diferencia lleva el signo del binomio.
Pregunta 3
¿Cuál es el resultado de (2x + 5)(2x − 5)?
✓ Correcto. Suma por diferencia: (2x)² − 5² = 4x² − 25. No hay términos medios (se cancelan).
✗ Incorrecto. Suma por diferencia: (2x+5)(2x−5) = (2x)²−5² = 4x²−25 (opción B). No es +25 (eso sería sin el signo diferencia). El resultado D sería de (2x−5)².
Pregunta 4
Factoriza: x² − 16
✓ Correcto. x²−16 = x²−4² es diferencia de cuadrados → (x+4)(x−4). No es un cuadrado de binomio porque no hay término medio.
✗ Incorrecto. x²−16 = x²−4²: diferencia de cuadrados → (x+4)(x−4) (opción C). (x±4)² incluiría el término ±8x, que aquí no está.
Pregunta 5
Factoriza: x² − 6x + 9
✓ Correcto. a²−2ab+b² con a=x, b=3: −2·x·3 = −6x ✓ y 3²=9 ✓. Resultado: (x−3)². El signo negativo del doble producto revela que es cuadrado de diferencia.
✗ Incorrecto. x²−6x+9: el término medio −6x = −2·3·x indica b=3 con signo negativo → (x−3)² (opción B). Si fuera (x+3)², el término medio sería +6x.
Pregunta 6
Desarrolla: (x − 2)(x + 7)
✓ Correcto. a=−2, b=7. Suma: (−2)+7 = +5 → coeficiente de x es +5. Producto: (−2)·7 = −14. Resultado: x²+5x−14.
✗ Incorrecto. (x−2)(x+7): suma de los no comunes = −2+7 = +5 (positivo), producto = −2·7 = −14. Resultado: x²+5x−14 (opción A).
Pregunta 7
Calcula mentalmente: 51²
✓ Correcto. 51² = (50+1)² = 50² + 2·50·1 + 1² = 2.500 + 100 + 1 = 2.601.
✗ Incorrecto. 51² = (50+1)² = 50²+2·50·1+1² = 2.500+100+1 = 2.601 (opción B). 2.500 sería 50² sin el doble producto ni el 1².
Pregunta 8
Factoriza: x² + 7x + 12
✓ Correcto. Buscar dos números que sumen 7 y multipliquen 12: 3 + 4 = 7 ✓ y 3 · 4 = 12 ✓. Resultado: (x+3)(x+4).
✗ Incorrecto. Busca dos números que sumen 7 y cuyo producto sea 12: son 3 y 4 (no 2 y 6 que suman 8, ni negativos). Resultado: (x+3)(x+4) (opción A).
Pregunta 9
¿Cuál es el resultado de simplificar: (a + b)² − (a − b)²?
✓ Correcto. (a+b)² = a²+2ab+b². (a−b)² = a²−2ab+b². Restando: (a²+2ab+b²)−(a²−2ab+b²) = 4ab. Los a² y b² se cancelan.
✗ Incorrecto. Desarrolla: (a²+2ab+b²)−(a²−2ab+b²) = 2ab+2ab = 4ab (opción B). 2ab solo si no restas correctamente el segundo cuadrado.
Pregunta 10
Si el área de un cuadrado de lado (x + 5) es 100 cm², ¿cuánto vale x?
✓ Correcto. Área = lado² → (x+5)² = 100 → x+5 = 10 (raíz positiva) → x = 5. Verifica: (5+5)² = 100 ✓.
✗ Incorrecto. (x+5)² = 100 → x+5 = √100 = 10 → x = 5 (opción A). Si dijeras x=10, el lado sería 15 y el área 225, no 100.

Memorízalos, identifícalos rápido y úsalos. Son herramientas que usarás durante toda la enseñanza media y la PAES.
