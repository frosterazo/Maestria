Potencias: Base racional y exponente entero

Nivel: 1° Año Medio
Eje: Números
Duración: 90 min
Currículum: MINEDUC 2026
OA 2
Objetivo de Aprendizaje: Comprender potencias de base racional y exponente entero, aplicando las propiedades de multiplicación y división de potencias, relacionándolas con el crecimiento y decrecimiento de cantidades, y resolviendo problemas de la vida diaria y otras asignaturas.
1
¿Qué es una potencia?

Una potencia es una multiplicación abreviada de un número por sí mismo. Tiene dos componentes:

Forma general
an
a = base · n = exponente
Base (a)

Es el número que se multiplica por sí mismo. Puede ser racional: entero, fracción o decimal.

Exponente (n)

Indica cuántas veces se multiplica la base por sí misma. Puede ser entero (positivo, negativo o cero).

Ejemplos
2³ = 2 · 2 · 2 = 8 (base 2, exponente 3)
(12)² = 12 · 12 = 14 (base racional, exponente positivo)
(0,5)² = 0,25 (base decimal)

💡Truco para exponente negativo
Cuando el exponente es negativo, "da vuelta" la fracción y el exponente se vuelve positivo. Es la regla más útil con fracciones.

5
Crecimiento y decrecimiento
Crecimiento (base > 1)

Si una población se duplica cada año, la cantidad después de n años es: P · 2ⁿ

Ej.: 1.000 bacterias se duplican cada hora. ¿Cuántas habrá tras 5 horas? → 1.000 · 2⁵ = 32.000

Decrecimiento (0 < base < 1)

Si una sustancia pierde la mitad cada hora, queda: M · (12)ⁿ

Ej.: 100 g de elemento radiactivo, vida media 1 año. Tras 4 años: 100 · (12)⁴ = 6,25 g

📈Aplicaciones reales
Las potencias aparecen en: interés compuesto bancario, crecimiento de bacterias, desintegración radiactiva, propagación de virus y depreciación de bienes.

Nivel Básico

Reconocimiento y cálculo simple

Ejercicio 1
Básico
¿Cuánto es (13)²?
✓ Correcto. (1/3)² = (1/3)·(1/3) = 1²/3² = 1/9. Con fracciones elevadas a una potencia: se elevan numerador y denominador por separado.
✗ Incorrecto. (1/3)² = 1²/3² = 1/9. No se multiplica el exponente por el numerador (eso daría 2/3), ni se multiplican numerador y denominador (1/6). La respuesta es 1/9 (opción A).
Ejercicio 2
Básico
¿Cuánto vale 5⁰?
✓ Correcto. Todo número distinto de cero elevado a 0 es igual a 1. Esto vale para cualquier base: 5⁰ = 1, (−3)⁰ = 1, (1/7)⁰ = 1.
✗ Incorrecto. El resultado no es 0, 5 ni −5. Todo número no nulo elevado a la potencia 0 es igual a 1. La respuesta es 1 (opción B).
Ejercicio 3
Básico
¿Cuál es el resultado de (−3)²?
✓ Correcto. (−3)² = (−3)·(−3) = +9. La base es −3 (con paréntesis) y el exponente par hace el resultado positivo. Negativo · negativo = positivo.
✗ Incorrecto. (−3)² = (−3)·(−3) = +9. El exponente es par, y negativo · negativo = positivo. La respuesta es 9 (opción B). Si fuera −3² (sin paréntesis), el resultado sería −9.
Nivel Medio

Aplicación de propiedades

Ejercicio 4
Medio
Calcula: 2³ · 2⁴
✓ Correcto. Misma base (2), producto: se suman los exponentes. 3 + 4 = 7. Resultado: 2⁷ = 128.
✗ Incorrecto. Cuando las bases son iguales y se multiplica, los exponentes se suman: 2³ · 2⁴ = 2³⁺⁴ = 2⁷ (opción A). Los exponentes no se multiplican.
Ejercicio 5
Medio
Calcula: (2³)²
✓ Correcto. Potencia de potencia: se multiplican los exponentes. 3 · 2 = 6. Resultado: 2⁶ = 64.
✗ Incorrecto. (aᵐ)ⁿ = aᵐ·ⁿ. Aquí (2³)² = 2³·² = 2⁶ (opción B). No se suman (eso daría 2⁵), ni se mantiene la base 4.
Ejercicio 6
Medio
¿Cuánto vale (25)⁻²?
✓ Correcto. Exponente negativo en fracción: se invierte la fracción y el exponente se vuelve positivo. (2/5)⁻² = (5/2)² = 25/4.
✗ Incorrecto. Con exponente negativo en una fracción: se invierte la fracción y el exponente pasa a positivo. (2/5)⁻² = (5/2)² = 25/4 (opción A). No es negativo ni queda sin elevar.
Nivel Avanzado

Aplicación a la realidad

Ejercicio 7
Avanzado
Una bacteria se duplica cada 20 minutos. Si comienzo con 10 bacterias, ¿cuántas tendré después de 2 horas?
✓ Correcto. 2 horas = 120 minutos. Con duplicación cada 20 min → 120 ÷ 20 = 6 períodos. Cantidad: 10 · 2⁶ = 10 · 64 = 640 bacterias.
✗ Incorrecto. 2 horas = 120 minutos. Duplicación cada 20 min → 6 períodos. Cantidad: 10 · 2⁶ = 10 · 64 = 640 (opción C). Error frecuente: usar 2 horas como exponente directamente.
Ejercicio 8
Avanzado
Simplifica: 3⁵ · 3⁻²3⁴
✓ Correcto. Numerador: 3⁵ · 3⁻² = 3⁵⁺⁽⁻²⁾ = 3³. Luego 3³ ÷ 3⁴ = 3³⁻⁴ = 3⁻¹ = 1/3.
✗ Incorrecto. Paso a paso: numerador 3⁵ · 3⁻² = 3³. Luego 3³ ÷ 3⁴ = 3⁻¹ (opción A). Exponente −1 significa 1/3, no 3.
Ejercicio 9
Avanzado
Calcula: (−12)³ · (−12)²
✓ Correcto. Misma base (−1/2), sumar exponentes: 3 + 2 = 5. Resultado: (−1/2)⁵. Exponente impar + base negativa = negativo. (−1)⁵/2⁵ = −1/32.
✗ Incorrecto. Igual base, se suman exponentes: (−1/2)³ · (−1/2)² = (−1/2)⁵. Exponente 5 impar + base negativa → resultado negativo. (−1/2)⁵ = −1/32 (opción A).
Ejercicio 10
Avanzado
Un auto vale $8.000.000 y cada año pierde la mitad de su valor. ¿Cuánto valdrá tras 3 años?
✓ Correcto. Decrecimiento: 8.000.000 · (1/2)³ = 8.000.000 ÷ 8 = 1.000.000. (1/2)³ = 1/8 porque se divide a la mitad 3 veces.
✗ Incorrecto. Fórmula: valor inicial · (1/2)ⁿ. → 8.000.000 · (1/2)³ = 8.000.000 / 8 = 1.000.000 (opción C). No es 4.000.000 (solo un año) ni 2.000.000 (dos años).

Pregunta 1
¿Cuánto es 4⁻²?
✓ Correcto. 4⁻² = 1/4² = 1/16. Exponente negativo = recíproco de la potencia positiva. No hace el resultado negativo.
✗ Incorrecto. 4⁻² = 1/4² = 1/16 (opción B). El exponente negativo no significa resultado negativo, sino recíproco: 1 dividido por la potencia positiva.
Pregunta 2
El resultado de (−1)²⁰²⁵ es:
✓ Correcto. 2025 es impar. (−1) elevado a un exponente impar es −1; a uno par es +1. Resultado: −1.
✗ Incorrecto. (−1) elevado a un exponente par = 1; a uno impar = −1. Como 2025 es impar, (−1)²⁰²⁵ = −1 (opción B).
Pregunta 3
Calcula: 2⁵ ÷ 2³
✓ Correcto. Misma base (2), cociente: restar exponentes. 5 − 3 = 2. Resultado: 2² = 4.
✗ Incorrecto. Cociente de igual base: se restan los exponentes. 2⁵ ÷ 2³ = 2⁵⁻³ = 2² (opción A). No se suman (eso sería el producto), ni se multiplican.
Pregunta 4
Simplifica: (34)² · (34)³
✓ Correcto. Misma base (3/4), producto: 2 + 3 = 5. Resultado: (3/4)⁵.
✗ Incorrecto. Igual base, producto: se suman exponentes. 2 + 3 = 5 → (3/4)⁵ (opción A). No se multiplican (eso daría 6) ni se elevan ambas fracciones.
Pregunta 5
Calcula: (53)⁻¹
✓ Correcto. Exponente −1 invierte la fracción. (5/3)⁻¹ = 3/5. Es el recíproco del número original.
✗ Incorrecto. (5/3)⁻¹ = 3/5 (opción B). El exponente −1 invierte la fracción: numerador y denominador se intercambian. No cambia el signo a negativo.
Pregunta 6
¿A qué es igual −3²?
✓ Correcto. Sin paréntesis, solo el 3 está elevado: 3² = 9. El signo − está afuera: −(3²) = −9. Diferente a (−3)² = 9.
✗ Incorrecto. Sin paréntesis, la potencia aplica solo al 3: −3² = −(3²) = −9 (opción B). Solo (−3)² = 9. Esta diferencia de paréntesis es crucial en exámenes.
Pregunta 7
Una población de bacterias se duplica cada hora. Si parto con 200, ¿cuántas hay tras 6 horas?
✓ Correcto. Crecimiento: 200 · 2⁶ = 200 · 64 = 12.800 bacterias. No se multiplica 200 · 6 = 1.200 (eso sería crecimiento lineal, no exponencial).
✗ Incorrecto. Crecimiento exponencial: 200 · 2⁶ = 200 · 64 = 12.800 (opción C). El error más común es multiplicar por 6 en vez de elevar al exponente 6.
Pregunta 8
El resultado de (2³ · 2⁻¹)² es:
✓ Correcto. Primero interior: 2³ · 2⁻¹ = 2³⁺⁽⁻¹⁾ = 2². Luego potencia de potencia: (2²)² = 2²·² = 2⁴.
✗ Incorrecto. Paso a paso: (2³ · 2⁻¹) = 2² → (2²)² = 2⁴ (opción A). No sumes todos los exponentes juntos ignorando la estructura del paréntesis.
Pregunta 9
El número 0,0001 expresado como potencia de 10 es:
✓ Correcto. 0,0001 = 1/10.000 = 1/10⁴ = 10⁻⁴. Cuenta los ceros después del punto decimal para determinar el exponente negativo.
✗ Incorrecto. 0,0001 = 1/10.000. Como 10.000 = 10⁴, entonces 1/10⁴ = 10⁻⁴ (opción B). Cuenta los lugares después de la coma: 0,_0_0_0_1 → 4 posiciones → exponente −4.
Pregunta 10
Una sustancia radiactiva pierde la mitad de su masa cada año. Si tengo 64 g, ¿cuánto queda después de 4 años?
✓ Correcto. 64 · (1/2)⁴ = 64 / 16 = 4 g. (1/2)⁴ = 1/16. Verificación: 64→32→16→8→4 (4 pasos de mitad).
✗ Incorrecto. Masa final: 64 · (1/2)⁴ = 64/16 = 4 g (opción C). 16 g sería tras 2 años; 32 g tras 1 año. Traza el camino: 64→32→16→8→4.

Resumen — Lo que aprendiste hoy

Las potencias son una herramienta poderosa para representar la multiplicación repetida. Aparecen en el interés bancario, el crecimiento de poblaciones y la desintegración de elementos radiactivos.

Tres claves: cuidado con los signos, dominar las propiedades, y entender qué modelan las potencias en la realidad.
