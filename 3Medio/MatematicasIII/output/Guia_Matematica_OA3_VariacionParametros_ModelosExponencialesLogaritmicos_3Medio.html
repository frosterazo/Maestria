Variar parámetros para ajustar modelos

Autor: Paulo Figueroa

OA 33° MedioParámetrosAjuste
OA 3Variar parámetros para ajustar un modelo exponencial o logarítmico según la situación.
1Qué es un parámetro

Un parámetro es un número dentro del modelo que controla su forma o posición. A diferencia de la variable, que cambia libremente, el parámetro se fija para representar una situación específica.

Por ejemplo, en \(N(t)=800(1,15)^t\), el número 800 indica el valor inicial y 1,15 indica el factor de crecimiento. Si cambiamos esos números, la curva cambia y el modelo describe otra situación.

Valor inicial
Desde dónde parte el modelo.
Tasa o base
Qué tan rápido crece o decrece.
Desplazamiento
Desde cuándo o desde dónde se mide.
Escala
Qué tan grande es el efecto.
Meta de la guía: comprender qué hace cada parámetro y cómo ajustarlo para que el modelo represente mejor la situación.

Ajustar crecimiento y decrecimiento

Teoría 2TasaFactor
1De tasa a factor

Cuando el parámetro se expresa como porcentaje, primero se convierte a factor multiplicativo. Este paso es fundamental para ajustar correctamente el modelo.

SituaciónTasaFactorModelo
Crecimiento 8%\(r=0,08\)\(1+r=1,08\)\(N(t)=N_0(1,08)^t\)
Crecimiento 35%\(r=0,35\)\(1+r=1,35\)\(N(t)=N_0(1,35)^t\)
Disminución 12%\(r=0,12\)\(1-r=0,88\)\(N(t)=N_0(0,88)^t\)
Ejemplo 1: ajustar por nueva tasa

Enunciado. Un modelo inicial era \(N(t)=1000(1,05)^t\). Luego se observa que el crecimiento real está más cerca de 9% por periodo. Ajusta el modelo.

Datos

  • \(N_0=1000\) (cantidad inicial).
  • Nueva tasa: \(r=0,09\).
  • Factor nuevo: \(1+r=1,09\).

Modelo ajustado: \(N(t)=1000(1,09)^t\).

Interpretación: no se cambió el valor inicial, solo el ritmo de crecimiento.

Advertencia: cambiar la tasa puede alterar mucho proyecciones lejanas. Un pequeño cambio en \(b\) se acumula con el tiempo.

Parámetros en modelos logarítmicos

Teoría 3LogarítmicoEscala
1Modelo base
\[ y=a+b\ln(x) \]
ParámetroQué controlaInterpretación
\(a\)Altura base.Desplaza el modelo hacia arriba o abajo.
\(b\)Intensidad del efecto logarítmico.Si \(b\) es mayor, el cambio inicial es más fuerte.
\(x\)Variable positiva.Debe cumplirse \(x>0\).
Ejemplo 2: cambiar la intensidad del aprendizaje

Modelo A: \(P(h)=400+60\ln(h)\). Modelo B: \(P(h)=400+90\ln(h)\). Ambos parten del mismo valor base, pero B aumenta más por cada unidad logarítmica.

Para \(h=5\): \(\ln(5)\approx1,609\).

Modelo A: \(400+60(1,609)\approx496,5\).

Modelo B: \(400+90(1,609)\approx544,8\).

Conclusión: aumentar \(b\) eleva la proyección para el mismo valor de \(h\).

Interpretación: en un modelo logarítmico, \(b\) no es una tasa porcentual. Es un coeficiente que amplifica o reduce el efecto de \(\ln(x)\).

Desplazamientos y restricciones

Teoría 4TraslaciónDominio
1Modelos con desplazamiento horizontal

Un modelo como \(y=a+b\ln(x-c)\) no empieza a tener sentido desde \(x>0\), sino desde \(x>c\). El parámetro \(c\) desplaza el inicio del modelo.

\[ y=a+b\ln(x-c)\qquad \Rightarrow \qquad x-c>0 \Rightarrow x>c \]
Ejemplo 3: adaptar un modelo al inicio real

Enunciado. Un modelo de rendimiento empieza a ser válido después de 3 días de adaptación. Se propone \(R(d)=50+20\ln(d-3)\). Explica el rol del parámetro 3.

Argumento del logaritmo: \(d-3\).

Condición matemática: \(d-3>0\).

Resultado: \(d>3\).

Interpretación: el parámetro 3 desplaza el inicio del modelo; antes del día 3 no se debe usar.

Subir \(a\): aumenta todas las proyecciones.
Aumentar \(c\): desplaza el inicio del modelo hacia la derecha.
Cambio de parámetroQué se ve en la curvaQué significa en contexto
\(a\) aumentaLa curva queda más arriba.Hay mayor rendimiento base.
\(b\) aumentaLa curva sube más rápido al inicio.Cada aumento de \(x\) tiene más impacto.
\(c\) aumentaEl dominio comienza más tarde.Existe un umbral antes de aplicar el modelo.

Ajustar un modelo a una situación

Ejemplo desafianteAjusteComparación
Ejemplo 4: escoger el mejor ajuste

Enunciado. Una población inicial es de 2000 personas. Los datos sugieren crecimiento cercano a 4% anual, pero una propuesta usa 9%. Compara ambos modelos para 5 años.

Datos del problema

  • \(N_0=2000\) (personas).
  • Modelo A: \(N_A(t)=2000(1,04)^t\).
  • Modelo B: \(N_B(t)=2000(1,09)^t\).
  • \(t=5\) (años).

Modelo A: \(N_A(5)=2000(1,04)^5\approx2000(1,217)=2433,3\).

Modelo B: \(N_B(5)=2000(1,09)^5\approx2000(1,539)=3077,2\).

Diferencia: \(3077,2-2433,3\approx643,9\) personas.

Conclusión: cambiar la tasa de 4% a 9% produce una proyección mucho mayor.

Interpretación: el parámetro debe elegirse con evidencia. Una tasa demasiado alta puede llevar a decisiones exageradas sobre recursos, presupuesto o infraestructura.
PreguntaRespuesta con el modelo
¿Qué parámetro cambió?El factor de crecimiento: \(1,04\) versus \(1,09\).
¿Qué se mantuvo fijo?El valor inicial \(N_0=2000\) personas y el tiempo \(t=5\) años.
¿Por qué importa?Porque una tasa mayor se multiplica cinco veces, no se suma una sola vez.

Ejercicios guiados

Ejercicios guiadosPaso a paso
Ejercicio guiado 1

Enunciado. Un modelo \(N(t)=700(1,08)^t\) debe ajustarse a una tasa de 12% sin cambiar el valor inicial. Escribe el nuevo modelo.

Datos

  • \(N_0=700\).
  • Nueva tasa \(r=0,12\).
  • Factor \(1+r=1,12\).

Reemplazo del parámetro: se cambia \(1,08\) por \(1,12\).

Modelo ajustado: \(N(t)=700(1,12)^t\).

Ejercicio guiado 2

Enunciado. En \(P(h)=350+70\ln(h)\), se quiere aumentar el efecto del estudio sin cambiar el valor base. ¿Qué parámetro se debe modificar?

Valor base: \(a=350\).

Efecto logarítmico: \(b=70\).

Decisión: para aumentar el efecto de \(\ln(h)\), se aumenta \(b\).

Respuesta: modificar el parámetro \(b\).

Solucionario y cierre

SolucionesPAES
1Respuestas explicadas
E1. En \(N(t)=300(1,2)^t\), el valor inicial es el coeficiente que multiplica la potencia. Además, \(N(0)=300(1,2)^0=300\). Respuesta correcta: A.
E2. Para crecimiento, el factor se calcula como \(1+r\). Con 15%, \(r=0,15\), factor \(1,15\). Con 6%, \(r=0,06\), factor \(1,06\). Respuesta correcta: A.
E3. En \(y=80+25\ln(x-4)\), el número 4 está dentro del argumento. Como \(x-4>0\), se obtiene \(x>4\). Además desplaza el inicio del modelo hacia la derecha. Respuesta correcta: A.
Clave final: ajustar parámetros no es cambiar números al azar. Cada cambio debe responder a evidencia o a una característica concreta de la situación.
Frase modelo: “Se modifica el parámetro ___ porque representa ___. Al aumentarlo/disminuirlo, el modelo cambia en ___, por lo tanto la proyección ___”.
