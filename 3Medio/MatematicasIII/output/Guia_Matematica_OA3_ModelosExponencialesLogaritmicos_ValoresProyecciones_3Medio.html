Modelos exponenciales y logarítmicos para proyectar

Autor: Paulo Figueroa

OA 33° MedioCrecimientoProyecciones
OA 3Utilizar modelos de situaciones de crecimiento y decrecimiento que involucran funciones exponencial y logarítmica para determinar valores o hacer proyecciones.
1Qué significa modelar

Un modelo matemático es una forma ordenada de describir una situación real con una expresión. No es una “adivinanza”: usa datos, supuestos y una regla para estimar valores o proyectar qué podría ocurrir después.

En esta guía trabajaremos dos familias de modelos. Los modelos exponenciales sirven cuando una cantidad se multiplica por un factor cada cierto tiempo. Los modelos logarítmicos sirven cuando una cantidad crece o cambia rápido al principio, pero luego cada aumento adicional produce un efecto menor.

Meta de la guía: aprender a usar modelos ya dados para calcular valores, hacer proyecciones y explicar qué significa el resultado en contexto.
Valor calculado: se obtiene reemplazando una entrada conocida en el modelo. Ejemplo: calcular \(N(4)\).
Proyección: estima un valor futuro suponiendo que el patrón se mantiene. Siempre debe interpretarse con cautela.

Recordatorio: potencias, tasas y logaritmos

Teoría 1BaseExponenteLogaritmo
1Lo mínimo que hay que recordar

Una potencia como \(2^5\) significa multiplicar 2 por sí mismo 5 veces. En un modelo exponencial, el exponente suele representar tiempo, ciclos o periodos.

IdeaSignificadoEjemplo
BaseNúmero que se repite como factor.En \(3^4\), la base es 3.
ExponenteIndica cuántas veces se aplica el factor.En \(3^4\), el exponente es 4.
Tasa de crecimientoPorcentaje que aumenta una cantidad por periodo.12% mensual.
Tasa de decrecimientoPorcentaje que disminuye una cantidad por periodo.8% semanal.
De porcentaje a factor:
Si aumenta 12%, el factor es \(1+0,12=1,12\).
Decrecimiento:
Si disminuye 8%, el factor es \(1-0,08=0,92\).
2Qué hace un logaritmo

Un logaritmo responde una pregunta inversa a una potencia: “¿a qué exponente debo elevar una base para obtener cierto número?”. Por ejemplo, \(\log_2(8)=3\), porque \(2^3=8\).

\[ \log_b(x)=y \quad \Longleftrightarrow \quad b^y=x \]
Conexión importante: si el modelo exponencial ayuda a encontrar una cantidad futura, el logaritmo puede ayudar a encontrar el tiempo necesario para alcanzar una cantidad.

Modelo exponencial de crecimiento

Teoría 2CrecimientoProyección
1Forma del modelo

Un modelo exponencial de crecimiento se usa cuando una cantidad se multiplica por un mismo factor en cada periodo. No crece sumando siempre lo mismo, sino multiplicando.

\[ N(t)=N_0(1+r)^t \]
SímboloQué representaUnidad típica
\(N(t)\)Cantidad proyectada después de \(t\) periodos.personas, bacterias, visitas, pesos
\(N_0\)Cantidad inicial, cuando \(t=0\).misma unidad de \(N(t)\)
\(r\)Tasa de crecimiento escrita como decimal.sin unidad
\(t\)Cantidad de periodos transcurridos.días, meses, años, semanas
Ejemplo conceptual: crecimiento por factor

Si una cuenta tiene 500 seguidores y crece 10% mensual, después de un mes no se suman 10 seguidores, sino el 10% de 500. Luego el siguiente mes se calcula sobre la cantidad nueva. Por eso el crecimiento se acelera.

Mes 0: \(500\) seguidores.

Mes 1: \(500\cdot1,10=550\) seguidores.

Mes 2: \(550\cdot1,10=605\) seguidores.

Modelo directo: \(N(2)=500(1,10)^2=605\).

Interpretación: en crecimiento exponencial, el aumento de cada periodo depende del valor acumulado anterior.

Ejemplo simple: proyección de crecimiento

Ejemplo simpleExponencialCrecimiento
Ejemplo 1: seguidores de una campaña escolar

Enunciado. Una campaña escolar en redes sociales comienza con 800 seguidores y crece un 15% mensual. Estima cuántos seguidores tendrá después de 4 meses si se mantiene el mismo patrón.

Datos del problema

  • \(N_0=800\) (seguidores iniciales).
  • \(r=0,15\) (tasa de crecimiento mensual).
  • \(t=4\) (meses).
  • \(N(4)\) = cantidad proyectada después de 4 meses.

Fórmula utilizada

\[ N(t)=N_0(1+r)^t \]

Se usa esta fórmula porque el crecimiento es porcentual y se aplica mes a mes sobre la cantidad acumulada.

Reemplazo y desarrollo

Reemplazo: \(N(4)=800(1+0,15)^4\).

Simplificar factor: \(1+0,15=1,15\).

Potencia: \(1,15^4\approx1,749\).

Multiplicación: \(800\cdot1,749\approx1399,2\).

Resultado final: \(N(4)\approx1399\) seguidores.

Interpretación: si la tendencia se mantiene, la campaña podría llegar a cerca de 1400 seguidores en 4 meses. Como se trata de personas, se redondea a un número entero.
Advertencia de proyección: el modelo no asegura que la campaña tendrá exactamente 1399 seguidores. Proyecta bajo el supuesto de que el crecimiento mensual de 15% se mantiene durante los 4 meses.

Modelo exponencial de decrecimiento

Teoría 3DecrecimientoFactor menor que 1
1Cuando la cantidad disminuye por porcentaje

Un modelo exponencial de decrecimiento se usa cuando una cantidad pierde un mismo porcentaje en cada periodo. El factor queda entre 0 y 1, porque la cantidad se va reduciendo.

\[ N(t)=N_0(1-r)^t \]
Ejemplo 2: batería de un dispositivo

Enunciado. Una batería tiene 100% de carga y pierde 18% de su carga cada hora de uso. Estima la carga después de 5 horas.

Datos del problema

  • \(N_0=100\) (% de carga inicial).
  • \(r=0,18\) (tasa de pérdida por hora).
  • \(t=5\) (horas).
  • \(N(5)\) = carga proyectada después de 5 horas.

Fórmula utilizada

\[ N(t)=N_0(1-r)^t \]

Desarrollo paso a paso

Reemplazo: \(N(5)=100(1-0,18)^5\).

Factor de conservación: \(1-0,18=0,82\). Esto significa que queda 82% de la carga cada hora.

Potencia: \(0,82^5\approx0,371\).

Multiplicación: \(100\cdot0,371=37,1\).

Resultado final: \(N(5)\approx37,1\%\).

Interpretación. Después de 5 horas, la batería tendría aproximadamente 37% de carga si se mantiene el mismo patrón de consumo.

Clave visual: si el factor es mayor que 1, hay crecimiento. Si el factor está entre 0 y 1, hay decrecimiento.

Usar logaritmos para encontrar tiempos

Teoría 4LogaritmosTiempo objetivo
1Cuando la incógnita está en el exponente

A veces no queremos saber cuánta cantidad habrá después de cierto tiempo, sino cuánto tiempo se necesita para alcanzar una cantidad. En ese caso, el tiempo está en el exponente y necesitamos logaritmos.

\[ N(t)=N_0b^t \qquad \Longrightarrow \qquad t=\frac{\log\left(\frac{N(t)}{N_0}\right)}{\log(b)} \]

La base del logaritmo puede ser 10 o \(e\), siempre que se use la misma base arriba y abajo. Lo importante es que el logaritmo permite “bajar” el exponente.

Ejemplo 3: ¿cuándo se duplican los seguidores?

Enunciado. Una cuenta crece 12% mensual. ¿Cuántos meses necesita para duplicar su cantidad inicial?

Datos del problema

  • \(b=1,12\) (factor mensual de crecimiento).
  • \(\frac{N(t)}{N_0}=2\) (duplicar significa llegar al doble).
  • \(t\) = meses necesarios.

Desarrollo

Modelo relativo: \(\frac{N(t)}{N_0}=1,12^t\).

Objetivo: \(2=1,12^t\).

Aplicar logaritmos: \(t=\frac{\log(2)}{\log(1,12)}\).

Cálculo: \(t\approx\frac{0,3010}{0,0492}\approx6,12\).

Resultado final: necesita cerca de 6,1 meses.

Interpretación. Si la tasa se mantiene, la cuenta duplicaría sus seguidores un poco después de 6 meses.

Modelo logarítmico: crecimiento que se desacelera

Teoría 5LogarítmicoCrecimiento lento
1Forma del modelo logarítmico

Un modelo logarítmico puede describir situaciones donde al inicio hay grandes avances, pero después cuesta más mejorar. Esto aparece en aprendizaje, percepción de sonido, experiencia de usuario o rendimiento al practicar.

\[ y=a+b\ln(x) \]
SímboloQué representaComentario
\(y\)Valor que se estima.puntaje, nivel, rendimiento, satisfacción
\(x\)Variable explicativa positiva.tiempo, horas, intentos; debe cumplirse \(x>0\)
\(a\)Valor base del modelo.ajusta altura de la curva
\(b\)Intensidad del cambio logarítmico.si \(b>0\), crece; si \(b<0\), decrece
Ejemplo 4: práctica y puntaje

Enunciado. Un modelo estima el puntaje de práctica mediante \(P(h)=420+85\ln(h)\), donde \(h\) es la cantidad de horas de estudio. Estima el puntaje para \(h=6\).

Datos del problema

  • \(P(h)=420+85\ln(h)\) (modelo de puntaje).
  • \(h=6\) (horas de estudio).
  • \(P(6)\) = puntaje estimado.

Desarrollo

Reemplazo: \(P(6)=420+85\ln(6)\).

Valor logarítmico: \(\ln(6)\approx1,792\).

Multiplicación: \(85\cdot1,792\approx152,32\).

Suma: \(420+152,32=572,32\).

Resultado final: \(P(6)\approx572\) puntos.

Interpretación. Según el modelo, 6 horas de estudio se asocian a un puntaje aproximado de 572 puntos. No significa certeza, sino una estimación.

Elegir entre modelo exponencial y logarítmico

Teoría 6SelecciónContexto
1La forma del cambio orienta el modelo
SituaciónModelo probablePor qué
Población que aumenta por porcentaje constante.Exponencial creciente.La variación depende del valor acumulado.
Precio de un objeto que pierde cierto porcentaje por año.Exponencial decreciente.Se conserva una fracción cada periodo.
Aprendizaje con mejoras rápidas al inicio y lentas después.Logarítmico creciente.El aumento marginal disminuye.
Percepción de intensidad o satisfacción que se satura.Logarítmico.Cada aumento adicional produce menos efecto.
Exponencial
Pregunta típica: ¿cuánto habrá después de \(t\) periodos si aumenta o disminuye por porcentaje?
Logarítmico
Pregunta típica: ¿qué valor se estima cuando el efecto crece cada vez más lento?
Cuidado con las proyecciones: un modelo puede ser útil dentro de cierto rango, pero no necesariamente funciona para tiempos muy lejanos. Una proyección siempre depende del supuesto “si la tendencia se mantiene”.
Mini ejemplo de selección

Enunciado. Una plataforma aumenta sus usuarios 9% mensual. ¿Qué modelo conviene para proyectar usuarios en 6 meses?

Dato relevante: aumenta por porcentaje mensual.

Razón: cada mes se multiplica por \(1,09\).

Modelo adecuado: exponencial creciente.

Ejercicios guiados

Ejercicios 1 de 3GuiadosPaso a paso
Ejercicio guiado 1: crecimiento exponencial

Enunciado. Una comunidad tiene 1200 integrantes y crece 6% anual. Estima la cantidad después de 3 años.

Datos

  • \(N_0=1200\) (integrantes).
  • \(r=0,06\) (crecimiento anual).
  • \(t=3\) (años).

Fórmula: \(N(t)=N_0(1+r)^t\).

Reemplazo: \(N(3)=1200(1,06)^3\).

Potencia: \(1,06^3\approx1,191\).

Multiplicación: \(1200\cdot1,191\approx1429,2\).

Resultado: aproximadamente 1429 integrantes.

Ejercicio guiado 2: modelo logarítmico

Enunciado. Un modelo de rendimiento está dado por \(R(h)=300+100\ln(h)\). Estima \(R(5)\).

Datos

  • \(R(h)=300+100\ln(h)\) (modelo).
  • \(h=5\) (horas).

Reemplazo: \(R(5)=300+100\ln(5)\).

Logaritmo: \(\ln(5)\approx1,609\).

Multiplicación: \(100\cdot1,609=160,9\).

Suma: \(300+160,9=460,9\).

Resultado: \(R(5)\approx461\).

E2. Datos: \(N_0=500\), \(r=0,10\), \(t=2\). Fórmula: \(N(t)=N_0(1+r)^t\). Reemplazo: \(N(2)=500(1+0,10)^2=500(1,10)^2\). Respuesta correcta: B.
