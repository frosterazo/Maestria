Elementos de la función cuadrática: el vértice

Autor: Paulo Figueroa

📖 Teoría Vértice Máximo / mínimo
OA 3 Identificar los elementos de las funciones cuadráticas: vértice, intersecciones con los ejes, eje de simetría y concavidad.
1¿Qué es el vértice?
Vértice: es el punto extremo de la parábola. Si la parábola abre hacia arriba, el vértice es el punto más bajo (mínimo). Si abre hacia abajo, es el punto más alto (máximo). Se anota \(V = (h, k)\).

El vértice siempre está sobre el eje de simetría. Su coordenada \(x\) es justamente \(h = -b/(2a)\), y su coordenada \(y\) se obtiene evaluando: \(k = f(h)\).

\(V = (h, k)\)   donde   \(h = -\dfrac{b}{2a}\),   \(k = f(h)\)
Interpretación práctica: el vértice responde dos preguntas importantes:
  • "¿Cuál es el valor máximo (o mínimo) que toma la función?" → Respuesta: \(k\).
  • "¿En qué valor de \(x\) ocurre ese máximo (o mínimo)?" → Respuesta: \(h\).
2Cómo calcularlo desde la forma general

Si tienes \(f(x) = ax^{2} + bx + c\), sigue estos dos pasos:

  1. Calcula \(h = -\dfrac{b}{2a}\).
  2. Evalúa \(k = f(h) = a h^{2} + b h + c\).
Ejemplo guiado
Problema Encontrar el vértice de \(f(x) = x^{2} - 6x + 10\).
\(a = 1\), \(b = -6\), \(c = 10\). \(h = -\dfrac{-6}{2 \cdot 1} = \dfrac{6}{2} = 3\). \(k = f(3) = 3^{2} - 6 \cdot 3 + 10 = 9 - 18 + 10 = 1\). Vértice: \(V = (3, 1)\). Como \(a=1>0\), es un mínimo.
\(f(x) = x^{2} - 6x + 10\): vértice mínimo \(V(3,\,1)\); abre hacia arriba.
3Cómo leerlo desde la forma canónica

Si la función está en forma canónica \(f(x) = a(x - h)^{2} + k\), el vértice se lee directamente:

\(f(x) = a(x - h)^{2} + k\)   ⇒   \(V = (h, k)\)
Ejemplo
Problema ¿Cuál es el vértice de \(f(x) = -2(x - 4)^{2} + 7\)?
Comparando con \(a(x - h)^{2} + k\): \(a = -2\), \(h = 4\), \(k = 7\). \(V = (4, 7)\). Como \(a < 0\), es un máximo.
Cuidado con los signos: en la forma canónica el valor de \(h\) tiene el signo CONTRARIO al que aparece dentro del paréntesis. Si ves \((x + 3)^{2}\), entonces \(h = -3\) (porque \(x + 3 = x - (-3)\)).
Metacognición
Sin avanzar: ¿cuál es el vértice de \(f(x) = (x + 1)^{2} - 5\)? Si llegas a \(V = (-1, -5)\), ya internalizaste cómo leer la forma canónica.

Intersecciones con los ejes

📖 Teoría Eje y · eje x Discriminante
1Intersección con el eje y

El eje y está formado por todos los puntos con \(x = 0\). Para encontrar dónde la parábola lo cruza, simplemente evaluamos la función en \(x = 0\).

\(f(0) = a \cdot 0^{2} + b \cdot 0 + c = c\)   ⇒   Intersección: \((0, c)\)

Por eso el coeficiente \(c\) es tan importante: siempre es el valor donde la parábola corta el eje y. Es el elemento más fácil de identificar.

Ejemplos directos
FunciónIntersección eje y
\(f(x) = 2x^{2} + 3x - 7\)\((0, -7)\)
\(f(x) = -x^{2} + 5\)\((0, 5)\)
\(f(x) = x^{2}\)\((0, 0)\)
\(f(x) = 3x^{2} - 6x\)\((0, 0)\)
2Intersecciones con el eje x

El eje x está formado por todos los puntos con \(y = 0\). Las intersecciones con el eje x son las soluciones de la ecuación \(ax^{2} + bx + c = 0\), también llamadas raíces o ceros de la función.

Hay tres métodos para encontrarlas:

Método 1: Factorización Cuando se puede factorizar fácilmente como \((x - r_{1})(x - r_{2})\).
Método 2: Fórmula cuadrática \(x = \dfrac{-b \pm \sqrt{b^{2} - 4ac}}{2a}\)
Método 3: Discriminante \(\Delta = b^{2} - 4ac\) predice cuántas raíces hay (0, 1 o 2).
¿Cuántas raíces tiene una parábola? Depende del discriminante \(\Delta = b^{2} - 4ac\):
  • Si \(\Delta > 0\): dos raíces reales distintas. La parábola corta el eje x en dos puntos.
  • Si \(\Delta = 0\): una raíz (doble). La parábola toca el eje x en su vértice (tangente).
  • Si \(\Delta < 0\): ninguna raíz real. La parábola no toca el eje x.
Ejemplo 1 — Dos raíces (factorización)
Problema Encontrar las raíces de \(f(x) = x^{2} - 5x + 6\).
Discriminante: \(\Delta = (-5)^{2} - 4 \cdot 1 \cdot 6 = 25 - 24 = 1 > 0\). Hay dos raíces. Factorizando: \(x^{2} - 5x + 6 = (x - 2)(x - 3)\). \((x - 2)(x - 3) = 0 \Rightarrow x = 2\) o \(x = 3\). Intersecciones eje x: \((2, 0)\) y \((3, 0)\).
\(f(x) = x^{2} - 5x + 6\): dos raíces reales (Δ > 0).
Ejemplo 2 — Sin raíces reales
Problema ¿Tiene raíces \(f(x) = x^{2} + 2x + 5\)?
\(\Delta = 2^{2} - 4 \cdot 1 \cdot 5 = 4 - 20 = -16 < 0\). No tiene raíces reales: la parábola no toca el eje x.
\(f(x) = x^{2} + 2x + 5\): no corta el eje x (Δ < 0).

Eje de simetría, concavidad, dominio y recorrido

📖 Teoría Simetría Dominio · recorrido
1Eje de simetría
Eje de simetría: es la recta vertical que divide la parábola en dos mitades exactamente iguales (una es el reflejo de la otra). Pasa siempre por el vértice.
Ecuación del eje de simetría: \(x = -\dfrac{b}{2a}\)

Observa que es la misma coordenada \(x\) del vértice. El eje y el vértice están "atados".

Ejemplo
Problema Encontrar el eje de simetría de \(f(x) = 2x^{2} - 8x + 5\).
\(a = 2\), \(b = -8\). \(x = -\dfrac{-8}{2 \cdot 2} = \dfrac{8}{4} = 2\). Eje de simetría: \(x = 2\).
2Concavidad

La concavidad indica hacia dónde "abre" la parábola. Se decide únicamente por el signo de \(a\):

Si \(a > 0\) Cóncava hacia arriba (↑). El vértice es un mínimo.
Si \(a < 0\) Cóncava hacia abajo (↓). El vértice es un máximo.

Una parábola con \(a > 0\) parece una "U"; con \(a < 0\) parece una "U invertida" (∩).

3Dominio y recorrido

El dominio de una función es el conjunto de valores de \(x\) que la función acepta. Para cualquier función cuadrática, podemos elevar al cuadrado, multiplicar y sumar sin restricciones, por lo que:

Dominio: \(\mathbb{R}\) (todos los números reales)

El recorrido (o rango) es el conjunto de valores que toma \(f(x)\). Para una parábola, el recorrido depende de la concavidad y del vértice:

CasoRecorridoExplicación
\(a > 0\) (abre arriba)\([k, +\infty)\) o \(y \geq k\)\(k\) es el valor mínimo
\(a < 0\) (abre abajo)\((-\infty, k]\) o \(y \leq k\)\(k\) es el valor máximo
Ejemplo
Problema Determinar el dominio y recorrido de \(f(x) = -x^{2} + 4x - 1\).
Dominio: \(\mathbb{R}\) (siempre). Vértice: \(h = -4/(2 \cdot -1) = 2\); \(k = f(2) = -4 + 8 - 1 = 3\). \(V = (2, 3)\). Como \(a = -1 < 0\), abre hacia abajo y \(k = 3\) es el máximo. Recorrido: \(y \leq 3\) (todos los valores menores o iguales a 3).
Metacognición
Cuando puedes mirar una parábola y decir inmediatamente: "abre arriba, vértice en (2, -3), entonces el recorrido es \(y \geq -3\)" — sin hacer ningún cálculo más — significa que has integrado todos los elementos en una imagen mental coherente.

Lectura inversa y aplicaciones del vértice

📖 Teoría Del gráfico a la fórmula Optimización
1De un gráfico a la fórmula

A veces tienes un gráfico (o un problema) y debes encontrar la fórmula de la parábola. La estrategia depende de qué información se conoce:

Conoces vértice + un punto Usa forma canónica \(f(x) = a(x - h)^{2} + k\). Reemplaza \((h, k)\) y usa el punto para encontrar \(a\).
Conoces las dos raíces + un punto Usa forma factorizada \(f(x) = a(x - r_{1})(x - r_{2})\). Reemplaza raíces y usa el punto para encontrar \(a\).
Conoces tres puntos Usa forma general \(f(x) = ax^{2}+bx+c\). Tres ecuaciones, tres incógnitas.
Ejemplo guiado — raíces + punto
Problema Una parábola pasa por \((1, 0)\), \((3, 0)\) y \((0, 6)\). Encontrar su fórmula.
Como pasa por \((1, 0)\) y \((3, 0)\), sus raíces son \(r_{1} = 1\) y \(r_{2} = 3\). Usamos forma factorizada: \(f(x) = a(x - 1)(x - 3)\). Usamos el tercer punto \((0, 6)\) para encontrar \(a\): \(6 = a(0 - 1)(0 - 3) = a \cdot (-1)(-3) = 3a\). De donde \(a = 2\). Fórmula: \(f(x) = 2(x - 1)(x - 3) = 2(x^{2} - 4x + 3) = 2x^{2} - 8x + 6\). \(f(x) = 2x^{2} - 8x + 6\)
\(f(x) = 2x^{2} - 8x + 6\) pasando por \((1,0)\), \((3,0)\) y \((0,6)\).
2El vértice como óptimo: aplicaciones

En muchos problemas reales, la pregunta clave es cuál es el máximo (o mínimo) de alguna cantidad. Si la cantidad se modela con una parábola, la respuesta está en el vértice.

Aplicación 1 — Altura máxima de un proyectil
Problema La altura de una pelota lanzada al aire es \(h(t) = -5t^{2} + 30t\) (\(h\) en metros, \(t\) en segundos). ¿Cuál es la altura máxima y cuándo se alcanza?
\(a = -5\), \(b = 30\), \(c = 0\). Como \(a < 0\), la parábola abre hacia abajo y tiene un máximo. Tiempo del máximo: \(t = -b/(2a) = -30/(-10) = 3\) s. Altura máxima: \(h(3) = -5(9) + 30(3) = -45 + 90 = 45\) m. Altura máxima: 45 m, a los 3 segundos.
Aplicación 2 — Ingreso máximo de un negocio
Problema El ingreso diario de una empresa se modela con \(I(p) = -2p^{2} + 40p\), donde \(p\) es el precio del producto. ¿A qué precio se maximiza el ingreso? ¿Cuánto es el ingreso máximo?
\(a = -2\), \(b = 40\). Parábola con máximo (porque \(a<0\)). Precio óptimo: \(p = -40/(-4) = 10\). Ingreso máximo: \(I(10) = -2(100) + 40(10) = -200 + 400 = 200\). Precio óptimo: $10; ingreso máximo: $200.
Patrón universal: en cualquier problema de "máximo" o "mínimo" donde aparezca una parábola, el truco es el mismo: calcular el vértice. La coordenada \(h\) te dice cuándo (o dónde) ocurre el óptimo; la coordenada \(k\) te dice cuánto vale.

🎯 Estrategia recomendada

Correcto. La intersección con el eje y siempre es \((0, c)\). Aquí \(c = -7\).

Incorrecto. Olvida el signo negativo de \(c\).
3Básico
Si el discriminante \(\Delta < 0\), la parábola:
A Corta el eje x en 2 puntos
Incorrecto. Eso ocurre cuando \(\Delta > 0\).
B Corta el eje x en 1 punto
Incorrecto. Eso ocurre cuando \(\Delta = 0\).
C No corta el eje x
Correcto. Cuando el discriminante es negativo, la ecuación \(ax^{2}+bx+c=0\) no tiene soluciones reales: la parábola no toca el eje x.
D Es una recta
Incorrecto. Una parábola sigue siendo parábola independientemente del discriminante.

Incorrecto. Error de signo en \(h\).
C \((6,\, 10)\)
Incorrecto. Confunde con los valores \(b\) y \(c\).

Incorrecto. Confunde con los valores \(b\) y \(c\).

Incorrecto. El discriminante es \(\Delta = 25 - 24 = 1 > 0\); sí hay 2 raíces.

Si \(f(x) = -2x^{2} + 8x - 6\), ¿cuál es el valor MÁXIMO?
A \(0\)
Incorrecto. No es el vértice; calcula \(k = f(h)\).
B \(2\)
Correcto. \(h = -8/(-4) = 2\); \(k = f(2) = -8 + 16 - 6 = 2\). Como \(a<0\), 2 es máximo.
C \(6\)
Incorrecto. Confunde con el valor absoluto de \(c\).
D \(-6\)
Incorrecto. Confunde con el valor de \(c\), pero el máximo es \(k\), no \(c\).

A \(y \geq -4\)
Correcto. \(h=2\), \(k = 4 - 8 = -4\). Como \(a>0\), el rango es \(y \geq k = -4\).
B \(y \geq 0\)
Incorrecto. El mínimo no es 0; es \(-4\) (vértice).
C \(y \leq -4\)
Incorrecto. Eso sería si abriera hacia abajo, pero \(a=1>0\) abre arriba.
D \(\mathbb{R}\)
Incorrecto. ℝ es el dominio, no el recorrido.

8Avanzado
Para que la parábola \(y = x^{2} + bx + 4\) toque tangencialmente al eje x, el valor de \(b\) debe ser:
A \(0\)
Incorrecto. Si \(b=0\), \(\Delta = -16 < 0\) y no toca el eje x.
B \(4\) o \(-4\)
Correcto. Tangencial significa \(\Delta = 0\): \(b^{2} - 16 = 0 \Rightarrow b = \pm 4\).
C \(2\)
Incorrecto. \(2^{2} - 16 = -12 < 0\), no es tangente.
D \(16\)
Incorrecto. Confunde el valor del discriminante con el de \(b\).

Correcto. Raíces 1 y 3, así que \(f(x) = a(x-1)(x-3)\). Usando \(f(0) = 6\): \(a \cdot (-1)(-3) = 6 \Rightarrow a = 2\).

Incorrecto. Esto da \(f(0) = 3\), no 6. Falta el factor \(a = 2\).

Incorrecto. Estas serían raíces \(-1\) y \(-3\), no \(1\) y \(3\).

Correcto. Cualquier número real se puede elevar al cuadrado y operar; el dominio es todo ℝ.

1
¿Cuál es el vértice de \(f(x) = (x + 2)^{2} - 5\)?
A \((2,\, -5)\)
Incorrecto. Olvida invertir el signo: \((x+2)^{2} = (x - (-2))^{2}\), \(h = -2\).
B \((-2,\, -5)\)
Correcto. En \(a(x-h)^{2}+k\): \(h = -2\), \(k = -5\).
C \((2,\, 5)\)
Incorrecto. Cambia signos de ambas coordenadas.
D \((-2,\, 5)\)
Incorrecto. Cambia el signo de \(k\).
2
¿Cuál es la intersección con el eje y de \(f(x) = -3x^{2} + 5\)?
A \((0,\, -3)\)
Incorrecto. \(-3\) es \(a\), no la intersección.
B \((0,\, 5)\)
Correcto. La intersección con eje y es \((0, c) = (0, 5)\).
C \((0,\, 0)\)
Incorrecto. Sería \((0,0)\) solo si \(c=0\).
D \((0,\, -5)\)
Incorrecto. Cambia el signo de \(c\) sin razón.
3
¿Cuántas intersecciones con el eje x tiene \(f(x) = x^{2} - 4x + 4\)?
A Ninguna
Incorrecto. \(\Delta = 16 - 16 = 0\), sí toca el eje x (en un solo punto).
B Una
Correcto. \(\Delta = 0\) significa una raíz doble. La parábola es tangente al eje x en su vértice.
C Dos
Incorrecto. Para dos raíces \(\Delta\) debe ser estrictamente positivo.
D Infinitas
Incorrecto. Una parábola tiene máximo 2 intersecciones con el eje x.
4
El eje de simetría de la parábola con vértice \((4, -2)\) es:
A \(y = -2\)
Incorrecto. El eje de simetría es vertical, escrito como \(x = ...\), no \(y = ...\).
B \(x = -2\)
Incorrecto. Toma la coordenada \(y\) del vértice, no \(x\).
C \(x = 4\)
Correcto. El eje de simetría pasa siempre por el vértice; es la recta vertical \(x = h = 4\).
D \(y = 4\)
Incorrecto. Confunde la dirección (eje vertical es \(x=...\)).
5
Si \(f(x) = 2x^{2} - 12x + 7\), el valor MÍNIMO es:
A \(7\)
Incorrecto. 7 es \(c\) (intersección con eje y), no el mínimo.
B \(-11\)
Correcto. \(h = 12/4 = 3\); \(k = f(3) = 18 - 36 + 7 = -11\). Como \(a>0\), es mínimo.
C \(-7\)
Incorrecto. Cambia el signo de \(c\) sin razón.
D \(12\)
Incorrecto. Confunde con el valor absoluto de \(b\).
6
¿Cuál de las siguientes parábolas tiene vértice en \((1, -4)\)?
A \(f(x) = (x - 1)^{2} - 4\)
Correcto. Forma canónica con \(h = 1\) (signo invertido en paréntesis), \(k = -4\).
B \(f(x) = (x + 1)^{2} - 4\)
Incorrecto. Daría vértice en \((-1, -4)\).
C \(f(x) = (x - 1)^{2} + 4\)
Incorrecto. Daría vértice en \((1, 4)\).
D \(f(x) = (x + 1)^{2} + 4\)
Incorrecto. Daría vértice en \((-1, 4)\).
7
Si la parábola \(y = x^{2} + bx + 9\) tiene su vértice sobre el eje x, entonces \(b\) es:
A \(0\)
Incorrecto. Con \(b=0\) el vértice sería \((0, 9)\), arriba del eje x.
B \(6\) o \(-6\)
Correcto. Vértice sobre eje x ⇔ \(\Delta = 0 \Rightarrow b^{2} - 36 = 0 \Rightarrow b = \pm 6\).
C \(9\)
Incorrecto. Confunde con el valor de \(c\).
D \(3\)
Incorrecto. \(3^{2} - 36 = -27 \neq 0\).
8
El recorrido (rango) de \(f(x) = -x^{2} + 6x - 5\) es:
A \(y \leq 4\)
Correcto. Vértice: \(h = -6/(-2) = 3\); \(k = -9 + 18 - 5 = 4\). Como \(a<0\), 4 es máximo. Rango: \(y \leq 4\).
B \(y \geq 4\)
Incorrecto. Con \(a<0\) la parábola abre hacia abajo; 4 es máximo, no mínimo.
C \(y \leq -5\)
Incorrecto. Confunde con el valor de \(c\), pero el límite del rango es \(k\), no \(c\).
D \(\mathbb{R}\)
Incorrecto. ℝ es el dominio, no el recorrido.
9
Para que \(f(x) = ax^{2} + 4x + 1\) NO tenga raíces reales, ¿qué valor de \(a\) se necesita?
A \(a > 4\)
Correcto. No raíces ⇔ \(\Delta < 0 \Rightarrow 16 - 4a < 0 \Rightarrow a > 4\).
B \(a < 4\)
Incorrecto. Invierte la desigualdad: con \(a < 4\) sí habría raíces.
C \(a > 0\)
Incorrecto. No basta con \(a > 0\); por ejemplo \(a=1\) sí tendría raíces.
D \(a = 0\)
Incorrecto. Con \(a=0\) no es cuadrática.
10
Una parábola tiene su vértice en el origen y abre hacia abajo. ¿Cuál podría ser su ecuación?
A \(y = x^{2}\)
Incorrecto. Abre hacia arriba, no hacia abajo.
B \(y = -x^{2}\)
Correcto. Vértice en \((0, 0)\) y \(a = -1 < 0\) ⇒ abre hacia abajo.
C \(y = x^{2} + 1\)
Incorrecto. Vértice en \((0, 1)\), no en el origen.
D \(y = -x^{2} - 3\)
Incorrecto. Vértice en \((0, -3)\), no en el origen.
