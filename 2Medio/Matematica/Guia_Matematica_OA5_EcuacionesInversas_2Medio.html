Ecuaciones de las funciones inversas

Autor: Paulo Figueroa

📖 Teoría Función inversa Metáfora de máquina
OA 5 Determinar las ecuaciones de las funciones inversas de funciones lineales y cuadráticas, comprendiendo el concepto mediante la metáfora de una máquina y la reflexión de la gráfica.
1La metáfora de la máquina

Imagina una función como una máquina: le entregas un número (la entrada \(x\)) y te devuelve otro número (la salida \(y = f(x)\)). Por ejemplo, la máquina "multiplica por 2 y suma 3" representa \(f(x) = 2x + 3\):

Entrada\(x = 4\)
→
Máquina \(f\)\(2x + 3\)
→
Salida\(y = 11\)

La función inversa, denotada \(f^{-1}\), es la máquina que hace lo contrario: le entregas la salida y te devuelve la entrada original.

Entrada\(y = 11\)
→
Máquina \(f^{-1}\)\(\dfrac{x - 3}{2}\)
→
Salida\(x = 4\)
Función inversa: dada una función \(f\), su inversa \(f^{-1}\) es aquella que "deshace" lo que hace \(f\). Si \(f\) transforma \(a \mapsto b\), entonces \(f^{-1}\) transforma \(b \mapsto a\).
Notación importante: \(f^{-1}\) NO significa \(\dfrac{1}{f}\). El "\(-1\)" indica inversa funcional, no exponente negativo. Es una convención que confunde, pero hay que recordarla.
2Propiedades fundamentales
Composición \(f(f^{-1}(x)) = x\)  y  \(f^{-1}(f(x)) = x\)
Intercambio de dominio/recorrido Dominio de \(f^{-1}\) = Recorrido de \(f\) (y viceversa).
Reflexión gráfica La gráfica de \(f^{-1}\) es la reflexión de \(f\) respecto a la recta \(y = x\).
Puntos intercambiados Si \((a, b) \in f\), entonces \((b, a) \in f^{-1}\).
3¿Toda función tiene inversa?

No. Para que una función tenga inversa, debe ser inyectiva (uno a uno): a cada salida le corresponde una única entrada. Si dos entradas distintas dan la misma salida, no se puede "deshacer" la operación sin ambigüedad.

Casos típicos en 2° Medio:
  • Funciones lineales \(f(x) = mx + n\) (con \(m \neq 0\)): siempre tienen inversa.
  • Funciones cuadráticas \(f(x) = ax^{2}+bx+c\): NO tienen inversa en todo \(\mathbb{R}\) (no son inyectivas). Para invertirlas, hay que restringir el dominio a una sola rama de la parábola.
Metacognición
Si te entregan \(f(x) = x^{2}\), te preguntan "¿qué entrada produjo la salida 9?". La respuesta es ambigua: pudo ser \(x = 3\) o \(x = -3\). Por eso \(x^{2}\) no tiene inversa única si su dominio es todo \(\mathbb{R}\). Restringiendo a \(x \geq 0\), la respuesta única es 3: \(f^{-1}(x) = \sqrt{x}\).

Inversa de una función lineal

📖 Teoría Funciones lineales Despejar x
1Método de los 3 pasos

Para encontrar la inversa de una función lineal \(f(x) = mx + n\), se sigue un procedimiento sistemático de tres pasos:

  1. Escribir la función como ecuación: reemplaza \(f(x)\) por \(y\). Queda \(y = mx + n\).
  2. Despejar \(x\): resolver para \(x\) en términos de \(y\). Queda \(x = \dfrac{y - n}{m}\).
  3. Intercambiar las variables: donde aparece \(y\) escribe \(x\) (y viceversa, si aplica). El resultado es \(f^{-1}(x) = \dfrac{x - n}{m}\).
¿Por qué se intercambian? Porque en la función inversa, el rol de entrada y salida se invierte. La salida original (\(y\)) pasa a ser la nueva entrada (que llamamos \(x\) por convención).
2Ejemplo guiado completo
Problema Encontrar la inversa de \(f(x) = 2x + 3\).
Paso 1: Escribir como ecuación: \(y = 2x + 3\). Paso 2: Despejar \(x\): \(y - 3 = 2x\) \(x = \dfrac{y - 3}{2}\) Paso 3: Intercambiar variables: \(f^{-1}(x) = \dfrac{x - 3}{2}\)
Verificación
\(f(f^{-1}(x)) = f\!\left(\dfrac{x - 3}{2}\right) = 2 \cdot \dfrac{x - 3}{2} + 3 = (x - 3) + 3 = x\) ✓ \(f^{-1}(f(x)) = f^{-1}(2x + 3) = \dfrac{(2x + 3) - 3}{2} = \dfrac{2x}{2} = x\) ✓ Las composiciones dan \(x\), confirmando que \(f^{-1}\) es la inversa correcta.
\(f(x) = 2x+3\) (púrpura), \(f^{-1}(x) = \tfrac{x-3}{2}\) (dorado) y la recta \(y=x\) (gris): son reflexiones una de otra.
3Fórmula general

Para cualquier función lineal \(f(x) = mx + n\) con \(m \neq 0\), la inversa es:

\(f^{-1}(x) = \dfrac{x - n}{m}\)

Memorízala: te ahorra rehacer los 3 pasos en cada problema. Aunque siempre conviene hacerlo al menos una vez para entender de dónde viene.

Caso especial \(m = 0\): si \(f(x) = n\) (constante), todos los puntos van al mismo valor \(n\). No es inyectiva (todas las entradas dan la misma salida), por lo tanto NO tiene inversa.

Inversa de una función cuadrática

📖 Teoría Restricción de dominio Raíz cuadrada
1El problema: la parábola no es inyectiva

La función \(f(x) = x^{2}\) asigna el mismo valor \(y\) a dos entradas opuestas: \(f(3) = 9\) y \(f(-3) = 9\). Por lo tanto, no es inyectiva y no tiene inversa en todo \(\mathbb{R}\).

Para poder invertirla, restringimos su dominio a una sola mitad de la parábola: o la rama derecha (\(x \geq 0\)) o la izquierda (\(x \leq 0\)). Lo mismo aplica a toda función cuadrática.

Rama derecha Dominio: \(x \geq h\) (donde \(h\) es la coordenada \(x\) del vértice). Inversa con signo "+".
Rama izquierda Dominio: \(x \leq h\). Inversa con signo "−".
2Método para cuadrática en forma canónica

Si la función está en forma canónica \(f(x) = a(x - h)^{2} + k\), el procedimiento es:

  1. Escribir \(y = a(x - h)^{2} + k\).
  2. Despejar \((x - h)^{2}\): \((x - h)^{2} = \dfrac{y - k}{a}\).
  3. Sacar raíz cuadrada (eligiendo el signo según la rama): \(x - h = \pm\sqrt{\dfrac{y - k}{a}}\).
  4. Despejar \(x\) e intercambiar variables: \(f^{-1}(x) = h \pm \sqrt{\dfrac{x - k}{a}}\).
Elección del signo:
  • Si el dominio se restringe a \(x \geq h\) (rama derecha): elegir el signo "+".
  • Si se restringe a \(x \leq h\) (rama izquierda): elegir el signo "−".
3Ejemplo guiado
Problema Encontrar la inversa de \(f(x) = (x - 3)^{2}\) con dominio restringido a \(x \geq 3\).
Paso 1: \(y = (x - 3)^{2}\). Paso 2: Sacar raíz cuadrada (con signo + porque \(x \geq 3\)): \(\sqrt{y} = x - 3\) Paso 3: Despejar \(x\): \(x = \sqrt{y} + 3\). Paso 4: Intercambiar variables: \(f^{-1}(x) = \sqrt{x} + 3\)

El dominio de \(f^{-1}\) es \(x \geq 0\) (que es el recorrido de \(f\) restringida).

\(f(x) = (x-3)^{2}\) con dominio \(x \geq 3\) (púrpura) y su inversa \(f^{-1}(x) = \sqrt{x} + 3\) (dorado), reflejadas en \(y = x\).
4Ejemplo con forma general
Problema Encontrar la inversa de \(f(x) = x^{2} + 4\) con dominio \(x \geq 0\).
\(y = x^{2} + 4 \Rightarrow x^{2} = y - 4\). \(x = \sqrt{y - 4}\) (signo + porque \(x \geq 0\)). Intercambiando: \(f^{-1}(x) = \sqrt{x - 4}\)

El dominio de \(f^{-1}\) es \(x \geq 4\) (el recorrido de \(f\) original, ya que el mínimo de \(f\) era \(k = 4\)).

Metacognición
Cada vez que veas una cuadrática para invertir, pregúntate dos cosas: "¿Cuál es el dominio restringido?" y "¿Cuál es el rango (recorrido) de \(f\)?". El recorrido de \(f\) es el dominio de \(f^{-1}\): bajo la raíz cuadrada, esos valores deben ser no negativos.

Verificación: composición y simetría

📖 Teoría Composición Reflexión
1Verificación por composición

La forma más rigurosa de comprobar que encontraste la inversa correcta es componer las dos funciones. Si \(f\) y \(f^{-1}\) son realmente inversas, entonces:

\(f(f^{-1}(x)) = x\)   y   \(f^{-1}(f(x)) = x\)

En palabras: aplicar primero una y luego la otra te debe devolver el valor original.

Ejemplo de verificación
Problema Verificar que \(f^{-1}(x) = \dfrac{x + 4}{3}\) es la inversa de \(f(x) = 3x - 4\).
\(f(f^{-1}(x)) = f\!\left(\dfrac{x + 4}{3}\right) = 3 \cdot \dfrac{x + 4}{3} - 4 = (x + 4) - 4 = x\) ✓ \(f^{-1}(f(x)) = f^{-1}(3x - 4) = \dfrac{(3x - 4) + 4}{3} = \dfrac{3x}{3} = x\) ✓ Ambas composiciones dan \(x\): confirmado que son inversas.
2Reflexión gráfica respecto a \(y = x\)

La reflexión gráfica es la forma más visual de entender la inversa. Si tomas la gráfica de \(f\) y la "doblas" sobre la recta \(y = x\), obtienes exactamente la gráfica de \(f^{-1}\).

¿Por qué \(y = x\)? Porque la inversa intercambia las coordenadas: el punto \((a, b)\) de \(f\) se convierte en \((b, a)\) en \(f^{-1}\). Y \((a, b)\) reflejado respecto a \(y = x\) es justamente \((b, a)\).
Visualización
\(f(x) = 3x - 4\) (púrpura) y \(f^{-1}(x) = \tfrac{x+4}{3}\) (dorado): son reflexiones respecto a \(y = x\) (gris).
3Casos especiales
Función "auto-inversa" Algunas funciones son su propia inversa: \(f = f^{-1}\). Ejemplos: \(f(x) = x\), \(f(x) = -x + a\) (para cualquier \(a\)).
Función identidad \(f(x) = x\) es su propia inversa, y su gráfica coincide con la recta \(y = x\).
Ejemplo de auto-inversa
Problema Demostrar que \(f(x) = -x + 5\) es su propia inversa.
Calculamos \(f^{-1}\) con el método estándar: \(y = -x + 5 \Rightarrow x = 5 - y\). Intercambiando: \(f^{-1}(x) = 5 - x = -x + 5\). \(f^{-1}(x) = f(x)\): la función es su propia inversa.
Metacognición
Las funciones auto-inversas son aquellas cuya gráfica es simétrica respecto a la recta \(y = x\). En el caso lineal, eso significa pendiente \(m = -1\) (o \(m = 1\) con \(n = 0\)). Identifícalas rápidamente sin hacer cálculos.

Incorrecto. Solo cambia el signo de \(n\), no aplica el procedimiento de inversa.

B \(f^{-1}(x) = -x + 5\)
Incorrecto. Cambia el signo de \(x\) sin justificación.
C \(f^{-1}(x) = -x - 5\)
Incorrecto. Cambia ambos signos sin justificación.

Incorrecto. Confunde con \(5 - x = -x + 5\), distinto a la inversa correcta.
3Básico
Si \(f(x) = 3x\), ¿cuánto vale \(f^{-1}(9)\)?
A \(3\)
Correcto. \(f^{-1}(x) = x/3\), así \(f^{-1}(9) = 3\). También: \(f(3) = 9\), por lo tanto \(f^{-1}(9) = 3\).
B \(9\)
Incorrecto. 9 es la entrada, no el resultado de aplicar la inversa.
C \(27\)
Incorrecto. Es \(f(9)\), no \(f^{-1}(9)\). Confunde direcciones.
D \(1/3\)
Incorrecto. Confunde \(f^{-1}\) con \(1/f\).

Incorrecto. Error de signo al despejar.

Incorrecto. Sería la inversa para \(x \leq 0\), no para \(x \geq 0\).

Incorrecto. Sería la misma función, no su inversa.

A \(f^{-1}(x) = \sqrt{x - 4}\)
Correcto. \(y = x^{2} + 4 \Rightarrow x^{2} = y - 4 \Rightarrow x = \sqrt{y - 4}\) (rama +). Intercambia variables.
B \(f^{-1}(x) = \sqrt{x} + 4\)
Incorrecto. Saca la raíz sin restar el 4 primero.

Incorrecto. Mismo error y restando.

B \(5\)
Correcto. Por propiedad fundamental, \(f(f^{-1}(x)) = x\) siempre. No hace falta calcular nada más: la respuesta es 5.
C \(9\)
Incorrecto. Calcula \(f(3)\) en lugar de aplicar la propiedad de composición.
D \(15\)
Incorrecto. No corresponde a ninguna composición válida.

Ecuaciones inversas — Preguntas PAES

🎯 Estilo PAES10 preguntasCon justificaciones
PAES — Funciones inversas: ecuaciones
1
La inversa de \(f(x) = 4x\) es:
A \(f^{-1}(x) = x - 4\)
Incorrecto. Resta en vez de dividir.
B \(f^{-1}(x) = \dfrac{x}{4}\)
Correcto. Para deshacer "multiplicar por 4", se divide por 4: \(y = 4x \Rightarrow x = y/4\).
C \(f^{-1}(x) = 4x\)
Incorrecto. Sería la misma función, no su inversa.
D \(f^{-1}(x) = -4x\)
Incorrecto. Cambia el signo sin justificación.
2
Si \(f(x) = \dfrac{x}{3} + 2\), su inversa es:
A \(f^{-1}(x) = 3x - 6\)
Correcto. \(y = x/3 + 2 \Rightarrow y - 2 = x/3 \Rightarrow x = 3(y-2) = 3y - 6\).
B \(f^{-1}(x) = 3x + 6\)
Incorrecto. Error de signo al multiplicar por 3.
C \(f^{-1}(x) = \dfrac{x}{3} - 2\)
Incorrecto. Solo cambia el signo de \(+2\), no aplica el procedimiento.
D \(f^{-1}(x) = \dfrac{3(x - 2)}{2}\)
Incorrecto. División incorrecta; sobra el 2 en denominador.
3
La gráfica de la función inversa \(f^{-1}\) es la reflexión de la gráfica de \(f\) respecto a:
A Eje x
Incorrecto. Reflejar respecto al eje x da la función \(-f(x)\), no la inversa.
B Eje y
Incorrecto. Reflejar respecto al eje y da \(f(-x)\), no la inversa.
C La recta \(y = x\)
Correcto. Como la inversa intercambia \((a, b) \leftrightarrow (b, a)\), la reflexión es respecto a la diagonal \(y = x\).
D El origen
Incorrecto. Reflejar respecto al origen da \(-f(-x)\), no la inversa.
4
Si \(f(x) = 2x + 1\) y \(g(x) = \dfrac{x - 1}{2}\), entonces \(g(x)\) es:
A La función inversa de \(f(x)\)
Correcto. Verifica: \(f(g(x)) = 2 \cdot (x-1)/2 + 1 = (x-1) + 1 = x\). Las composiciones dan \(x\), son inversas.
B Una función paralela a \(f(x)\)
Incorrecto. Tienen pendientes recíprocas (2 y 1/2), no iguales.
C Una función perpendicular
Incorrecto. Las pendientes de funciones perpendiculares son opuestas y recíprocas (2 y -1/2), no recíprocas.
D La misma función
Incorrecto. Son distintas: \(f(0) = 1\), \(g(0) = -1/2\).
5
¿Cuál es la inversa de \(f(x) = -x + 7\)?
A \(f^{-1}(x) = x - 7\)
Incorrecto. Error en el signo de \(x\).
B \(f^{-1}(x) = -x + 7\)
Correcto. Despejando: \(y = -x + 7 \Rightarrow x = -y + 7\). Intercambiando: \(f^{-1}(x) = -x + 7 = f(x)\). Es su propia inversa.
C \(f^{-1}(x) = -x - 7\)
Incorrecto. Cambia signo de \(n\) sin justificación.
D \(f^{-1}(x) = x + 7\)
Incorrecto. Cambia signo de \(x\) sin justificación.
6
Si \(f(x) = x^{2}\) para \(x \geq 0\), entonces \(f^{-1}(16)\) es:
A \(4\)
Correcto. \(f^{-1}(x) = \sqrt{x}\), entonces \(f^{-1}(16) = \sqrt{16} = 4\). Equivalente: ¿qué número positivo al cuadrado da 16? \(4^{2} = 16\).
B \(-4\)
Incorrecto. Con dominio restringido a \(x \geq 0\), solo se considera la rama positiva.
C \(16\)
Incorrecto. Repite el argumento sin aplicar la inversa.
D \(256\)
Incorrecto. Calcula \(f(16)\) en vez de \(f^{-1}(16)\).
7
La inversa de la función \(f(x) = ax + b\) (con \(a \neq 0\)) es:
A \(f^{-1}(x) = ax - b\)
Incorrecto. Solo cambia signo de \(b\); no es el procedimiento correcto.
B \(f^{-1}(x) = \dfrac{x - b}{a}\)
Correcto. Despeja: \(y = ax + b \Rightarrow y - b = ax \Rightarrow x = (y-b)/a\). Esa es la fórmula general.
C \(f^{-1}(x) = \dfrac{x}{a} + b\)
Incorrecto. Divide y suma \(b\), pero debe ser restado y todo dividido.
D \(f^{-1}(x) = \dfrac{x + b}{a}\)
Incorrecto. Error de signo en el numerador.
8
Si \(f(2) = 7\), entonces se cumple que:
A \(f^{-1}(2) = 7\)
Incorrecto. La inversa intercambia entrada y salida: \(f(2) = 7\) implica \(f^{-1}(7) = 2\), NO \(f^{-1}(2) = 7\).
B \(f^{-1}(7) = 2\)
Correcto. Si \(f(2) = 7\), la inversa "deshace" esta correspondencia: \(f^{-1}(7) = 2\).
C \(f^{-1}(2) = -7\)
Incorrecto. Confunde con el inverso aditivo.
D \(f(2) = f^{-1}(2)\)
Incorrecto. Esto solo sería cierto si la función fuera su propia inversa en el punto 2.
9
Si \(f(x) = x^{2} - 9\) con \(x \geq 0\), la inversa es:
A \(f^{-1}(x) = \sqrt{x + 9}\)
Correcto. \(y = x^{2} - 9 \Rightarrow x^{2} = y + 9 \Rightarrow x = \sqrt{y + 9}\) (rama +). Intercambia variables.
B \(f^{-1}(x) = \sqrt{x - 9}\)
Incorrecto. Error de signo al despejar.
C \(f^{-1}(x) = \sqrt{x} + 9\)
Incorrecto. El \(9\) debe ir DENTRO de la raíz.
D \(f^{-1}(x) = x^{2} + 9\)
Incorrecto. Eleva al cuadrado en vez de extraer raíz.
10
Para que una función tenga inversa, debe ser:
A Continua
Incorrecto. Hay funciones continuas que no son invertibles (por ejemplo \(x^{2}\) en todo \(\mathbb{R}\)).
B Inyectiva (uno a uno)
Correcto. A cada salida le corresponde una única entrada. Sin esa propiedad, la inversa sería ambigua.
C Sobreyectiva solamente
Incorrecto. Sobreyectiva sola no basta; debe ser inyectiva. (Biyectiva = inyectiva + sobreyectiva).
D Constante
Incorrecto. Las funciones constantes son las MENOS invertibles: todas las entradas dan el mismo valor.
