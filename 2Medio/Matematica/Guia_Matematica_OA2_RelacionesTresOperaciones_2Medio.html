Potencias, raíces y logaritmos: una sola familia

Autor: Paulo Figueroa

📖 Teoría Visión unificada Tres preguntas, un solo triángulo
OA 2 Mostrar que comprenden las relaciones entre potencias, raíces enésimas y logaritmos, estableciendo equivalencias y resolviendo problemas en contextos diversos.
1Tres operaciones, un mismo triángulo

Hasta ahora estudiaste tres operaciones que parecían independientes: la potencia, la raíz y el logaritmo. La verdad es que las tres son la misma idea, vista desde tres ángulos distintos. Comparten una misma ecuación de fondo:

\(b^{c} = a\)

Dependiendo de cuál de las tres letras conozcas y cuál te falte, tienes una operación distinta:

Lo que sabesLo que buscasOperaciónNotación
Base \(b\) y exponente \(c\)El valor \(a\)Potencia\(b^{c}\)
Valor \(a\) y exponente \(c\)La base \(b\)Raíz\(\sqrt[c]{a}\)
Base \(b\) y valor \(a\)El exponente \(c\)Logaritmo\(\log_{b}(a)\)
Imagen mental: piensa en un triángulo con tres vértices: base, exponente y valor. Cada vez que despejas uno de ellos a partir de los otros dos, estás usando una operación distinta. Pero la relación entre los tres siempre es la misma: \(b^{c} = a\).
2Un ejemplo desde tres ángulos

Considera la igualdad \(2^{3} = 8\). Esta única afirmación se puede leer de tres maneras distintas:

La ecuación \(2^{3} = 8\) responde tres preguntas

  • Pregunta de potencia: "¿Cuánto vale 2 elevado a 3?" → Respuesta: \(2^{3} = 8\).
  • Pregunta de raíz: "¿Qué número elevado al cubo da 8?" → Respuesta: \(\sqrt[3]{8} = 2\).
  • Pregunta de logaritmo: "¿A qué exponente hay que elevar 2 para obtener 8?" → Respuesta: \(\log_{2}(8) = 3\).

Las tres preguntas son distintas, pero las tres respuestas están conectadas por la misma ecuación. Si sabes cualquier dos de las tres letras, puedes encontrar la tercera usando la operación correspondiente.

Ejemplo guiado
Problema Sabiendo que \(\sqrt[3]{125} = 5\), escribe esta relación de las otras dos formas (potencia y logaritmo).
Forma raíz: \(\sqrt[3]{125} = 5\). Forma potencia (elevando ambos lados al cubo): \(5^{3} = 125\). Forma logaritmo (preguntando por el exponente): \(\log_{5}(125) = 3\). Las tres dicen lo mismo: \(b=5\), \(c=3\), \(a=125\).
3Las dos operaciones inversas de la potencia

La potencia es una operación "directa": tienes base y exponente, calculas el valor. Para "deshacer" esta operación hay dos caminos posibles, dependiendo de lo que conozcas:

¿Conoces el exponente? Para encontrar la base, usas raíz: \(b = \sqrt[c]{a}\)
¿Conoces la base? Para encontrar el exponente, usas logaritmo: \(c = \log_{b}(a)\)

Por eso decimos que la raíz y el logaritmo son "operaciones inversas" de la potencia. La diferencia entre ellas es qué dato faltaba.

Analogía: imagina una balanza con tres pesas: una a la izquierda (base), otra al medio (exponente) y la fórmula \(b^{c}=a\) en el plato derecho. Si te quitan la pesa izquierda, usas raíz para reconstruirla. Si te quitan la del medio, usas logaritmo. Si te quitan ambas, te falta información.
Metacognición
Sin avanzar: dada la igualdad \(3^{4} = 81\), escríbela como raíz y como logaritmo. Si llegaste a \(\sqrt[4]{81} = 3\) y \(\log_{3}(81) = 4\), ya internalizaste la relación. Esta capacidad de "traducir" entre los tres lenguajes es la clave del OA 2.

Relaciones inversas en profundidad

📖 Teoría Potencia ↔ Raíz Potencia ↔ Logaritmo
1Potencia y raíz: el puente del exponente fraccionario

Ya conociste esta relación en el OA 1. La equivalencia es:

\(\sqrt[n]{a^{m}} = a^{m/n}\)

La raíz no es una operación "diferente" de la potencia; es la misma operación pero con exponente fraccionario. El índice de la raíz pasa al denominador del exponente.

Ejemplos rápidos
\(\sqrt{a} = a^{1/2}\) \(\sqrt[3]{a^{2}} = a^{2/3}\) \(\sqrt[5]{x^{4}} = x^{4/5}\)

La relación es bidireccional: cada raíz se puede escribir como potencia y cada potencia con exponente fraccionario se puede escribir como raíz.

2Potencia y logaritmo: el puente del exponente como respuesta

Aquí está la nueva relación a profundizar. El logaritmo es la operación que "extrae el exponente" de una potencia.

\(\log_{b}(a) = c \iff b^{c} = a\)

Lee esta equivalencia con cuidado: el símbolo \(\iff\) significa "si y solo si". Las dos expresiones son completamente equivalentes: dicen exactamente lo mismo.

Forma logarítmicaForma exponencial (potencia)
\(\log_{2}(8) = 3\)\(2^{3} = 8\)
\(\log(1000) = 3\)\(10^{3} = 1000\)
\(\log_{5}(1) = 0\)\(5^{0} = 1\)
\(\log_{3}(81) = 4\)\(3^{4} = 81\)
\(\log_{2}(\sqrt{2}) = \tfrac{1}{2}\)\(2^{1/2} = \sqrt{2}\)
Ejemplo guiado
Problema Sabiendo que \(2^{10} = 1024\), calcular \(\log_{2}(1024)\).
Aplicando la equivalencia: \(\log_{2}(1024) = 10\). La potencia y el logaritmo "intercambian" la información: ahora el exponente es la respuesta. \(\log_{2}(1024) = 10\)
Cuándo se "naturalmente" pide cada operación:
  • Conoces el exponente y necesitas el valor → potencia ("cuánto se multiplica").
  • Conoces la base y necesitas el exponente → logaritmo ("cuántas veces se multiplica").
  • Conoces el exponente y necesitas la base → raíz ("qué número, al multiplicarse \(n\) veces, da el valor").
3Las dos relaciones, juntas

Veamos cómo conviven las tres operaciones en un solo problema.

Ejemplo guiado
Problema Calcular \(\log_{2}(\sqrt{8})\) usando ambas relaciones.
Paso 1 — convertir la raíz a potencia: \(\sqrt{8} = 8^{1/2}\). Paso 2 — reescribir 8 como potencia de 2: \(8 = 2^{3}\). Entonces \(8^{1/2} = (2^{3})^{1/2} = 2^{3/2}\). Paso 3 — aplicar logaritmo: \(\log_{2}(2^{3/2}) = 3/2\) (porque el exponente al que hay que elevar 2 para obtener \(2^{3/2}\) es… \(3/2\)). \(\log_{2}(\sqrt{8}) = \dfrac{3}{2}\)

Observa cómo las tres operaciones se complementan: empezamos con una raíz, la convertimos a potencia, y luego usamos logaritmo para extraer el exponente. Esta capacidad de "saltar" entre representaciones es lo que define a alguien que realmente entiende potencias, raíces y logaritmos.

Metacognición
Cuando ves un problema con cualquiera de las tres operaciones, pregúntate: "¿podría reescribir esto usando otra de las tres?". Esa flexibilidad mental es lo que separa al estudiante que recuerda propiedades del que comprende la relación.

Aplicaciones unificadas

📖 Teoría Resolver ecuaciones Modelos reales
1Despejar una incógnita: ¿qué operación uso?

Cuando enfrentas una ecuación con una incógnita, la operación que necesitas depende de dónde está la incógnita:

EcuaciónIncógnita está…Para despejar usa…
\(x^{3} = 27\)en la baseraíz cúbica: \(x = \sqrt[3]{27} = 3\)
\(2^{x} = 32\)en el exponentelogaritmo: \(x = \log_{2}(32) = 5\)
\(b^{4} = 81\)en la base (con exponente conocido)raíz cuarta: \(b = \sqrt[4]{81} = 3\)
\(\log_{3}(x) = 2\)como argumento del logpotencia: \(x = 3^{2} = 9\)
Regla mnemotécnica: "para deshacer hay que hacer lo opuesto". El opuesto de elevar es extraer raíz (si la incógnita es la base) o aplicar logaritmo (si la incógnita es el exponente).
2Aplicación: crecimiento exponencial y tiempo

Los modelos de crecimiento exponencial (poblaciones, bacterias, interés compuesto, virus) tienen la forma:

\(P(t) = P_{0} \cdot a^{t}\)

donde \(P_{0}\) es el valor inicial y \(a\) es el factor de crecimiento por unidad de tiempo. Si te preguntan cuánto vale P en cierto tiempo, usas potencia. Si te preguntan cuánto tiempo toma alcanzar cierto valor, usas logaritmo.

Ejemplo guiado
Problema Una colonia de bacterias se duplica cada hora. Si empiezas con 100 bacterias, ¿cuántas habrá después de 5 horas? ¿En cuánto tiempo llegarás a 1600 bacterias?
Modelo: \(P(t) = 100 \cdot 2^{t}\). Parte 1 (potencia): después de 5 horas, \(P(5) = 100 \cdot 2^{5} = 100 \cdot 32 = 3200\) bacterias. Parte 2 (logaritmo): queremos \(P(t) = 1600\). Entonces \(100 \cdot 2^{t} = 1600\), o sea \(2^{t} = 16\). Despejando con logaritmo: \(t = \log_{2}(16) = 4\) horas. 5 h → 3200 bacterias; para alcanzar 1600, t = 4 h.

Observa cómo en el mismo problema usamos potencia (para predecir el futuro) y logaritmo (para "preguntar al pasado": cuánto tiempo ha transcurrido).

3Aplicación: dimensiones geométricas

Cuando trabajas con áreas (cuadrados) o volúmenes (cubos), las tres operaciones aparecen naturalmente:

  • Área conocida, lado desconocido (cuadrado): \(\ell = \sqrt{A}\).
  • Volumen conocido, arista desconocida (cubo): \(a = \sqrt[3]{V}\).
  • Lado conocido, área desconocida: \(A = \ell^{2}\) (potencia).
  • Comparar tamaños relativos en órdenes de magnitud: usar logaritmos (cuántas "veces" más grande).
Ejemplo guiado
Problema Un cuadrado A tiene área de 100 m² y un cuadrado B tiene área de 10 000 m². ¿Cuántas veces más grande es el lado de B respecto al de A? ¿Y cuántos órdenes de magnitud difieren sus áreas?
Lado de A: \(\sqrt{100} = 10\) m. Lado de B: \(\sqrt{10000} = 100\) m. Razón de lados: \(100/10 = 10\). El lado de B es 10 veces el de A. Razón de áreas: \(10000/100 = 100\). En escala logarítmica: \(\log(100) = 2\) órdenes de magnitud. Lado B = 10 · lado A; áreas difieren en 2 órdenes de magnitud.
Metacognición
Antes de leer la próxima sección de cualquier problema PAES con potencias/raíces/logs, pregúntate: "¿qué letra del triángulo \(b^{c}=a\) está despejada y cuál falta?". Esa sola pregunta te orienta hacia la operación correcta sin tener que memorizar tablas.

Si \(2^{3} = 8\), ¿cuál es el logaritmo de 8 en base 2?
A \(2\)
Incorrecto. 2 es la base, no el exponente.
B \(3\)
Correcto. Por equivalencia: \(b^{c} = a \iff \log_{b}(a) = c\). El logaritmo es el exponente, que es 3.
C \(8\)
Incorrecto. 8 es el argumento, no el logaritmo.
D \(1\)
Incorrecto. 1 sería \(\log_{2}(2)\), no \(\log_{2}(8)\).
2Básico
Si \(\sqrt{16} = 4\), entonces \(4^{2}\) es igual a:
A \(8\)
Incorrecto. \(8 = 4 \cdot 2\), no \(4^{2}\).
B \(16\)
Correcto. La raíz es la inversa de la potencia: \(\sqrt{16} = 4 \iff 4^{2} = 16\).
C \(4\)
Incorrecto. \(4^{2} = 16\), no 4.
D \(64\)
Incorrecto. \(64 = 4^{3}\), exponente incorrecto.
3Básico
La operación inversa de "elevar al cubo" es:
A Raíz cuadrada
Incorrecto. La raíz cuadrada deshace "elevar al cuadrado", no al cubo.
B Raíz cúbica
Correcto. La raíz cúbica (\(\sqrt[3]{\ }\)) deshace "elevar al cubo": \((\sqrt[3]{a})^{3} = a\).
C Logaritmo en base 3
Incorrecto. El log en base 3 deshace potencias de base 3 (\(3^{x}\)), no la operación de elevar al cubo.
D Multiplicación por 3
Incorrecto. Multiplicar por 3 no es la inversa de elevar al cubo.
IntermedioEjercicios 4 – 7
4Intermedio
Si \(\log_{b}(x) = c\), ¿cuál es el equivalente en forma de potencia?
A \(b^{x} = c\)
Incorrecto. Intercambia los roles de \(x\) y \(c\).
B \(b^{c} = x\)
Correcto. Por definición: \(\log_{b}(x) = c \iff b^{c} = x\). La base se conserva, el logaritmo (\(c\)) es el exponente.
C \(c^{x} = b\)
Incorrecto. Intercambia base con logaritmo.
D \(c^{b} = x\)
Incorrecto. Cambia base por logaritmo en la potencia.
5Intermedio
Si \(\sqrt[3]{27} = 3\), ¿cuál es \(\log_{3}(27)\)?
A \(1\)
Incorrecto. \(\log_{3}(3) = 1\), no \(\log_{3}(27)\).
B \(2\)
Incorrecto. \(3^{2} = 9\), no 27.
C \(3\)
Correcto. \(\sqrt[3]{27} = 3 \iff 3^{3} = 27 \iff \log_{3}(27) = 3\). Los tres son equivalentes.
D \(27\)
Incorrecto. 27 es el argumento, no el resultado del logaritmo.
6Intermedio
Si \(\log_{5}(125) = 3\), entonces se cumple:
A \(5 \cdot 3 = 125\)
Incorrecto. \(5 \cdot 3 = 15\), no 125.
B \(5^{3} = 125\)
Correcto. Por definición, el logaritmo es el exponente al que hay que elevar la base.
C \(125 / 5 = 25\)
Incorrecto. La división no expresa la relación logaritmo-potencia.
D \(3^{5} = 243\)
Incorrecto. Intercambia la base con el exponente.
7Intermedio
Si \(2^{x} = 16\), entonces \(x\) es igual a:
A \(2\)
Incorrecto. \(2^{2} = 4\), no 16.
B \(4\)
Correcto. \(2^{4} = 16\), entonces \(x = \log_{2}(16) = 4\).
C \(8\)
Incorrecto. \(2^{8} = 256\), se pasa.
D \(16\)
Incorrecto. Confunde \(x\) (el exponente) con el valor de la potencia.
AvanzadoEjercicios 8 – 10
8Avanzado
¿Cuál de las siguientes afirmaciones describe correctamente las tres operaciones?
A Todas son la misma operación con distinto nombre
Incorrecto. Son tres operaciones distintas, aunque conectadas. La pregunta es qué se busca en cada caso.
B \(\sqrt{x} = x^{1/2}\), pero el logaritmo es totalmente distinto
Incorrecto. El logaritmo también está conectado: es la operación que extrae el exponente de una potencia.
C \(\sqrt{x} = x^{1/2}\) y el logaritmo es la inversa de la potencia (extrae el exponente)
Correcto. Las tres operaciones están conectadas por la ecuación \(b^{c} = a\): la raíz despeja la base, el logaritmo despeja el exponente.
D Las tres son operaciones independientes
Incorrecto. Toda la idea del OA 2 es que están conectadas — son tres formas de leer la misma ecuación.
9Avanzado
Si \(\log_{b}(256) = 4\), ¿cuál es \(b\)?
A \(16\) (porque \(b^{2} = 256\))
Incorrecto. Usa exponente 2 en lugar del exponente dado (4).
B \(4\) (porque \(b^{4} = 256\))
Correcto. La equivalencia es \(b^{4} = 256\). Despejando: \(b = \sqrt[4]{256} = 4\) (porque \(4^{4} = 256\)).
C \(64\) (porque \(b = 256/4\))
Incorrecto. La división no resuelve esta ecuación; necesitas raíz cuarta.
D \(260\) (porque \(b = 256 + 4\))
Incorrecto. La suma no tiene relación con esta operación.
10Avanzado
¿Cuál de las siguientes igualdades es FALSA?
A Si \(b^{c} = x\), entonces \(\log_{b}(x) = c\)
Verdadera. Es la definición misma del logaritmo.
B Si \(\sqrt{x} = y\), entonces \(y^{2} = x\)
Verdadera. Es la relación inversa entre raíz cuadrada y cuadrado.
C Si \(\log_{b}(x) = c\), entonces \(b/c = x\)
FALSA — esta es la respuesta correcta. Lo correcto es \(b^{c} = x\), no la división \(b/c\). Es un error frecuente confundir la operación inversa con división.
D Si \(x^{1/n} = y\), entonces \(y^{n} = x\)
Verdadera. Es la equivalencia entre raíz n-ésima y exponente fraccionario.

PAES — Relaciones entre potencias, raíces y logaritmos
1
¿Cuál es la operación inversa de \(\log_{2}(x)\)?
A Raíz cuadrada
Incorrecto. La raíz cuadrada deshace al cuadrado, no al logaritmo.
B Multiplicar por 2
Incorrecto. La multiplicación no deshace un logaritmo.
C Elevar 2 al exponente \(x\), es decir \(2^{x}\)
Correcto. El logaritmo y la potencia son operaciones inversas: si \(y = \log_{2}(x)\), entonces \(x = 2^{y}\).
D Dividir por 2
Incorrecto. Dividir es la inversa de multiplicar, no de logaritmo.
2
Si \(4^{x} = 64\), entonces \(x\) es igual a:
A \(3\)
Correcto. \(4^{3} = 64\), así que \(x = \log_{4}(64) = 3\).
B \(16\)
Incorrecto. 16 es \(64/4\); no resuelve la ecuación exponencial.
C \(4\)
Incorrecto. \(4^{4} = 256\), no 64.
D \(60\)
Incorrecto. 60 es \(64-4\); resta inválida en este contexto.
3
¿Cuál de las siguientes expresiones es equivalente a \(\log_{3}(81)\)?
A \(81/3 = 27\)
Incorrecto. La división no es la operación que define al logaritmo.
B \(3^{81}\)
Incorrecto. Intercambia argumento con exponente.
C El exponente al que hay que elevar 3 para obtener 81, que es 4
Correcto. Por definición, \(\log_{3}(81) = 4\) porque \(3^{4} = 81\).
D \(\sqrt{81} = 9\)
Incorrecto. La raíz cuadrada no calcula el logaritmo.
4
Si \(\log(x) = 3\) (base 10), entonces \(x\) es igual a:
A \(30\)
Incorrecto. 30 es \(10 \cdot 3\); confunde la operación.
B \(100\)
Incorrecto. 100 corresponde a \(\log(100) = 2\), no 3.
C \(1000\)
Correcto. Por equivalencia: \(\log(x) = 3 \iff 10^{3} = x = 1000\).
D \(0{,}001\)
Incorrecto. Eso correspondería a \(\log(x) = -3\), exponente negativo.
5
Si \(\sqrt[5]{32} = 2\), ¿cuál es \(\log_{2}(32)\)?
A \(2\)
Incorrecto. 2 es la base del logaritmo, no su valor.
B \(5\)
Correcto. \(\sqrt[5]{32} = 2 \iff 2^{5} = 32 \iff \log_{2}(32) = 5\). El índice de la raíz es el logaritmo.
C \(32\)
Incorrecto. 32 es el argumento del logaritmo, no su valor.
D \(1/5\)
Incorrecto. Eso sería el exponente fraccionario de \(\sqrt[5]{}\) como potencia, no el log de 32.
6
La relación entre raíz n-ésima y exponente fraccionario es:
A \(\sqrt[n]{a} = 1/a^{n}\)
Incorrecto. Confunde con exponente negativo.
B \(\sqrt[n]{a} = a^{1/n}\)
Correcto. El índice de la raíz pasa al denominador del exponente.
C \(\sqrt[n]{a} = na\)
Incorrecto. Multiplica base por índice, sin relación con la operación.
D \(\sqrt[n]{a} = a^{n}\)
Incorrecto. Eleva en lugar de extraer raíz.
7
Si \(\log_{b}(a) = c\), entonces:
A \(a + c = b\)
Incorrecto. La suma no es la operación inversa del logaritmo.
B \(b \cdot c = a\)
Incorrecto. La multiplicación tampoco es la inversa correcta.
C \(b^{c} = a\)
Correcto. Es la definición misma del logaritmo: el logaritmo es el exponente.
D \(c/b = a\)
Incorrecto. La división no expresa la relación log-potencia.
8
Una población se duplica cada año según \(P = P_{0} \cdot 2^{t}\). ¿Cuántos años deben pasar para que la población sea 8 veces la inicial?
A \(2\) años
Incorrecto. En 2 años: \(2^{2} = 4\), aún no 8.
B \(3\) años
Correcto. Quiero \(2^{t} = 8\), entonces \(t = \log_{2}(8) = 3\) años. Usar logaritmo para despejar el tiempo en un modelo exponencial.
C \(4\) años
Incorrecto. En 4 años la población sería 16 veces la inicial, no 8.
D \(8\) años
Incorrecto. Confunde el factor (8 veces) con el tiempo (años).
9
¿Cuál es el valor de \(\log_{2}(\sqrt{2})\)?
A \(1\)
Incorrecto. \(\log_{2}(2) = 1\), pero el argumento aquí es \(\sqrt{2}\), no 2.
B \(\dfrac{1}{2}\)
Correcto. \(\sqrt{2} = 2^{1/2}\), entonces \(\log_{2}(\sqrt{2}) = \log_{2}(2^{1/2}) = 1/2\). Une raíz, potencia y log.
C \(2\)
Incorrecto. Eso sería \(\log_{2}(4)\).
D \(\sqrt{2}\)
Incorrecto. Repite el argumento; el logaritmo es el exponente, no la raíz.
10
Si \(10^{x} = 0{,}001\), ¿cuál es \(x\)?
A \(3\)
Incorrecto. \(10^{3} = 1000\), no 0,001. Falta el signo negativo.
B \(-3\)
Correcto. \(0{,}001 = 1/1000 = 10^{-3}\), así que \(x = -3\).
C \(\dfrac{1}{3}\)
Incorrecto. Eso sería un exponente fraccionario; no corresponde a un decimal de la forma \(10^{-n}\).
D \(-\dfrac{1}{3}\)
Incorrecto. Combina dos errores: fraccionario y negativo sin justificación.
